# 場景：商店街轉角的販賣機——投一罐的錢，掉出兩罐
# 角色：桐原 美緒（きりはら みお，同校高一，別班，笑點低、愛佔小便宜嘴上說著、其實老實）
# 季節：九月（初秋，傍晚微涼）
# 出處：手寫・合世界聖經

```
（商店街轉角那台老販賣機，燈管一閃一閃的，賣的飲料一半都褪了色。傍晚的石板路上，人聲慢慢稀了。）
（我投了錢，按下彈珠汽水。咚——咚。掉下來的聲音，響了兩次。）

主角：「……欸？」

桐原美緒：「掉兩罐了吧！我剛剛也是！這台今天壞得很慷慨！」

（隔壁站著個女生，手裡已經抱著兩罐咖啡，笑得肩膀發抖。是桐原，別班的。）

主角：「妳也中了？」

桐原美緒：「嗯！投一罐的錢掉兩罐，這種好事一年碰不到一次。」

主角：「所以妳站在這，是想等它繼續掉？」

桐原美緒：「……被你看穿了。可是良心跟貪心在打架，我還沒打完。」

主角：「打成幾比幾了？」

桐原美緒：「貪心暫時領先。但你一來，良心好像多了個裁判。」

（她把多出來的那罐咖啡舉起來看了看，又看看店家亮著燈的方向，嘆了口氣。）

桐原美緒：「算了。多的這罐，我拿去還給裡面的老闆娘。你那罐呢？」

主角：「……那我也還。一起被裁判看著，賴不掉了。」

桐原美緒：「你這人也太容易被帶動。」

主角：「是妳先把良心搬出來的。」

桐原美緒：「哈哈，也對。不過——」

（她把自己那罐彈珠汽水塞進我手裡，玻璃珠在瓶口叮地響了一下。）

桐原美緒：「這罐是我正常投的，請你。當作陪我還飲料的跑腿費。」

主角：「還沒還就先發跑腿費，妳這帳算得真鬆。」

桐原美緒：「傍晚了嘛，算太清楚會餓。走啦，趁老闆娘還沒關店。」

（兩個人抱著多出來的罐子往店裡走，夕陽把販賣機的光壓成一條長長的影。）
```
