# 場景：商店街盡頭的舊唱片行——翻遍整箱黑膠，找一首只記得旋律的歌
# 角色：藤堂 奏（とうどう かなで，同校高一，別班，玩樂團、外冷內熱、聊到音樂就停不下來）
# 季節：十月（秋，午後）
# 出處：手寫・合世界聖經

```
（商店街盡頭那家舊唱片行，門一推開，鈴鐺和黴味一起迎上來。老唱機在角落轉著，沙沙的，像下雨。）
（我蹲在「特價一枚一百圓」的紙箱前，一張一張翻，找一首連名字都想不起來的歌。）

藤堂奏：「你那樣翻，翻到明年也找不到。」

主角：「……藤堂？」

（她抱著一疊黑膠站在我背後，別班的藤堂。學校裡她總是一副懶得說話的樣子，此刻眼神卻很亮。）

藤堂奏：「這箱沒分類，全靠手氣。你要找什麼？」

主角：「不知道名字。只記得旋律，B面第二首，很慢，有口琴。」

藤堂奏：「線索少得可憐。不過……口琴、慢歌、會被丟進一百圓箱——大概是哪個年代的地方樂團。」

主角：「妳光聽這幾個字就能猜？」

藤堂奏：「猜不準。但範圍能縮小。你哼幾句。」

主角：「……在店裡哼？」

藤堂奏：「怕什麼。老闆耳背，唱機又吵。哼。」

（我硬著頭皮哼了幾句走音的旋律。她歪著頭聽，手指在箱沿無意識地打拍子。）

藤堂奏：「等等——那個轉音，我好像聽過。是這張。」

（她從自己那疊裡抽出一張封套褪色的唱片，走去把老唱機的針，輕輕搭上去。）

藤堂奏：「B面第二首。你聽。」

（沙沙聲之後，慢慢的口琴響起來。就是它。我愣在原地，藤堂難得地笑了。）

主角：「……真的是這首。妳怎麼連別人記不住名字的歌都認得。」

藤堂奏：「因為好歌不需要被記住名字，也會被人一直哼下去。你剛剛不就證明了。」

主角：「這句話，很像妳會寫進歌詞裡的。」

藤堂奏：「……被你發現了。這張一百圓，你買，然後借我翻錄。這樣我們都不虧。」

主角：「成交。B面第二首，算我們今天一起挖到的。」

（口琴還在轉，午後的光透過落灰的玻璃，把整間店照得像一張舊照片。）
```
