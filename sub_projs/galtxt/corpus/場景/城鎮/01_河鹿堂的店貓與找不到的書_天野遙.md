# 場景：老書店「河鹿堂」——找一本書，卻先遇上店貓
# 角色：天野 遙（あまの はるか，同校高一，別班，聲音有點大、直來直往、其實怕生）
# 季節：五月（新綠，午後）
# 出處：手寫・合世界聖經

```
（放學繞來河鹿堂。日和橋那頭的水聲從半開的木門縫裡漏進來。）
（櫃檯後的老爺爺照例在打盹，收音機開著卻沒人在聽。我蹲下來，是因為書架最底層趴著一團橘白色的東西。）

（店裡的老貓——大家都叫牠「梅吉」，因為牠總睡在蓋梅花章的印台旁。）

主角：「……你又睡在這。擋到書了啦。」

天野遙：「牠不會讓的。我試過整整一個下午。」

（聲音從書架另一頭繞過來，比店裡任何東西都響。是天野。）

主角：「天野？妳也來找書？」

天野遙：「找《河童》。文庫那版。可是排到一半就被這隻毛球坐住了。」

主角：「所以妳剛剛在跟貓談判。」

天野遙：「牠先耍賴的。」

（她伸手要去搬貓，梅吉懶懶地睜一隻眼，又閉上。天野的手僵在半空，最後只是很輕地摸了摸牠的背。）

天野遙：「……算了。牠睡得那麼香。」

主角：「原來妳嗓門那麼大，對貓卻不敢用。」

天野遙：「吵到牠很失禮吧。牠比我早住進這家店。」

（我沒忍住笑了。她耳根有點紅，把視線甩去窗外那張壞了彈簧的皮沙發。）

天野遙：「你笑什麼。你不是也蹲在這半天不敢動。」

主角：「……被妳發現了。那本《河童》，等牠醒了我幫妳抽出來。」

天野遙：「你怎麼知道會在牠底下？」

主角：「猜的。爺爺說梅吉專挑好書坐。」

（爺爺不知何時醒了，在櫃檯後笑得像沒事人，拿起那枚小小的梅花章，在空氣裡虛蓋了一下。）

天野遙：「……那我等。反正末班車還早。」

主角：「嗯。我陪妳等牠。」
```
