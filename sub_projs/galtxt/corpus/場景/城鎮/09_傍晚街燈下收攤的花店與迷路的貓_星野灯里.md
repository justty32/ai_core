# 場景：傍晚的商店街街燈——花店收攤時分，一隻迷路的貓與一個不肯回家的女生
# 角色：星野 灯里（ほしの あかり，同校高一，同班，開朗但傍晚會突然安靜、家裡開花店）
# 季節：六月（初夏，梅雨的空檔）
# 出處：手寫・合世界聖經

```
（傍晚的商店街，街燈一盞一盞亮起來，把石板路照成一條溫溫的暖黃。店家陸續拉下鐵捲門。）
（轉角的花店還亮著，一個女生正把賣剩的花一桶一桶搬進屋。是同班的星野。她家開花店，我知道，但沒真的來過。）

星野灯里：「咦，是你？這個時間怎麼在這邊晃。」

主角：「散步。……妳在收攤？」

星野灯里：「嗯，傍晚的花不搬進去會蔫。你要不要幫我搬這桶？很重的。」

主角：「這桶是……向日葵？六月就有了？」

星野灯里：「溫室催的。我媽說，梅雨天最需要一點假裝出來的太陽。」

（我搬起那桶花，比想像中沉。星野在旁邊笑，笑著笑著，忽然安靜下來，望著街燈發呆。）

主角：「……怎麼了？」

星野灯里：「沒有。就是每次街燈亮的時候，我都會有點捨不得。白天過完了的感覺。」

主角：「妳白天不是笑得最大聲的那個嗎。」

星野灯里：「所以傍晚才要收一收啊。不然一直亮著，會累的。」

（就在這時，花桶底下鑽出一團灰影——一隻濕答答的小貓，喵地叫了一聲，蜷在燈影裡發抖。）

主角：「哪來的貓？」

星野灯里：「不知道，剛剛就一直在店門口繞。找不到回家的路吧。」

主角：「跟某個傍晚不想回家的人一樣。」

星野灯里：「……你是在說我嗎。」

主角：「猜的。妳搬花搬得那麼慢。」

（她被說中了，蹲下去把小貓捧進懷裡，用擦花的乾布替牠擦毛，動作很輕。）

星野灯里：「那今天就先讓牠住店裡。花店晚上有暖氣，牠不會冷。」

主角：「那妳呢？也該回家了。」

星野灯里：「嗯。有你陪我把最後一桶搬完，好像就捨得回去了。……謝謝喔。」

（小貓在她懷裡打了個呵欠。街燈把兩個人的影子，和一團小小的灰影，一起拉得長長的。）
```
