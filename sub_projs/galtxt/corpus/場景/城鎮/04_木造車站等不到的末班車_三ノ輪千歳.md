# 場景：老式木造車站——末班車前的月台，一個人不夠，兩個人剛好
# 角色：三ノ輪 千歳（みのわ ちとせ，同校高一，鄰班，話少、愛看時刻表、其實很怕一個人）
# 季節：十一月（深秋，入夜快）
# 出處：手寫・合世界聖經

```
（地方支線的木造車站，暖黃的燈把月台照成一小塊淺淺的島。站外已經全黑了。）
（電子看板壞了好幾年，末班車的時間只能看牆上那張手寫的時刻表。我盯著它，越看越沒把握。）

主角：「……欸，十八點四十七那班，是不是已經開走了？」

三ノ輪千歳：「開走了。剛剛那班就是。」

（聲音從長椅那頭傳來。是三ノ輪，鄰班的。她抱著書包，坐在燈影最邊上。）

主角：「三ノ輪同學？那……下一班是？」

三ノ輪千歳：「十九點二十一。末班。這站一天只剩兩班了。」

主角：「妳背下來了？」

三ノ輪千歳：「來這裡發呆的次數多，就記住了。」

（她說完又低下頭。我在她隔壁坐下，長椅涼得像鐵。遠處霞川的水聲混在風裡。）

主角：「妳常一個人等到這麼晚？」

三ノ輪千歳：「……嗯。家裡沒人等我回去，急也沒用。」

主角：「那更該早點回去吧，這月台冷得要命。」

三ノ輪千歳：「一個人的時候不覺得冷。是等到習慣了。」

（她把書包抱得更緊。我忽然不知道該不該接話，就把販賣機買的熱可可推過去一罐。）

主角：「這個給妳。反正我手也只有兩隻。」

三ノ輪千歳：「……為什麼給我。」

主角：「因為妳看時刻表看得比我認真，值一罐。」

（她愣了一下，指尖碰到罐子的熱，肩膀好像鬆了一點點。）

三ノ輪千歳：「……謝謝。那，末班車來之前，你先別走。」

主角：「本來就走不了，我也在等同一班。」

三ノ輪千歳：「那就好。今天的月台，好像沒那麼冷。」

（遠遠地，鐵軌傳來一點細細的震動。末班車的燈，在黑暗盡頭亮成一小點。）
```
