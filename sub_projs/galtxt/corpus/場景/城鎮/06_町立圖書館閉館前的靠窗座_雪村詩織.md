# 場景：町立圖書館——閉館音樂響起前，靠窗那個總被佔走的位子
# 角色：雪村 詩織（ゆきむら しおり，同校高一，圖書委員、輕聲細語、認真到有點固執）
# 季節：一月（隆冬，午後很快變黃昏）
# 出處：手寫・合世界聖經

```
（町立圖書館的午後，暖氣把整室的紙味都焙得暖暖的。靠窗那排座位，能看見霞川河堤上光禿禿的樹。）
（我照例佔了最角落那個位子——結果桌上已經擺著一本攤開的書和一支筆。人不在。）

主角：「……有人坐了啊。」

雪村詩織：「啊，那個位子……不好意思，是我的。書我忘了收。」

（抱著一疊要上架的書，她從書架後探出頭，聲音壓得很低，像怕吵醒整座圖書館。）

主角：「妳是圖書委員？難怪動作那麼熟。」

雪村詩織：「嗯。閉館前要把還回來的書歸位。這個位子……我每天都坐，習慣了。」

主角：「靠窗、看得到河堤、離暖氣最近。是好位子。」

雪村詩織：「你也懂。所以我更不想讓給別人……啊，不是那個意思。」

主角：「沒關係，我懂被搶座位的心情。那我坐妳旁邊那個？」

雪村詩織：「……可以。不過旁邊那個窗縫會漏風，冬天有點冷。」

主角：「妳連哪個座位漏風都知道。」

雪村詩織：「一整年都坐在這裡，會知道的。哪個燈會閃、哪塊地板會響、幾點鐘太陽會照到第三排——全都知道。」

（她說著說著，自己先不好意思地笑了，把懷裡的書抱得更緊。）

主角：「那妳算這座圖書館的活地圖了。」

雪村詩織：「才不是。只是……待得比別人久而已。」

（窗外的天色沉下來，館內的燈次第亮起。廣播輕輕響起閉館前的提示音樂。）

雪村詩織：「閉館音樂了。你要借的書，快去蓋章吧，我幫你留意櫃檯。」

主角：「妳呢？」

雪村詩織：「我最後走。要把靠窗這排的椅子都推回去……這是我的位子，也是我的收尾。」

主角：「那我陪妳推。兩個人，比較快。」

雪村詩織：「……謝謝。明天這個位子，還是我先到喔。」

主角：「行。明天我來搶第二好的那個。」

（她低頭笑了，把手裡最後一本書，輕輕插回它該在的地方。）
```
