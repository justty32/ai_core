# 場景：老書店「河鹿堂」後方私藏書庫——集滿梅花章，才被領進去的地方
# 角色：綾瀬 楓（あやせ かえで，同校高一，別班，安靜、常泡在書店、把書當老朋友）
# 季節：三月（早春，微寒）
# 出處：手寫・合世界聖經

```
（河鹿堂的後方，有一道總是半掩的木門，通往爺爺的私藏書庫。門上貼著一張泛黃的紙：集滿十枚梅花章者，方得入內。）
（我攢了半年，今天終於湊齊。爺爺瞇著眼數了數我卡片上的章，笑著把門推開。裡頭比外面更暗、更靜、書更舊。）

綾瀬楓：「……你也集滿了？我還以為，這半年只有我在蓋章。」

（書架深處，一個女生抬起頭。手裡捧著一本線裝書，是別班的綾瀬。她說話很輕，像怕驚動了滿室的書。）

主角：「綾瀬同學？妳早就進來過了？」

綾瀬楓：「上個月剛滿十枚。爺爺說，這裡的書不外借，只能在這張沙發上看。」

主角：「就是那張彈簧壞掉的皮沙發？」

綾瀬楓：「嗯。坐下去會陷進去，起來要花點力氣。可是靠窗，光最好。看書的話，是全店最好的位子。」

主角：「妳講得像這裡是妳家。」

綾瀬楓：「……某種程度上是。這些書，比我認識的很多人都久。翻開它們，就像跟老朋友重新見面。」

主角：「所以妳集章，不是為了炫耀，是為了進來見這些老朋友。」

綾瀬楓：「你這樣講，好像有點難為情。……但沒說錯。」

（她把手裡那本書挪了挪，在沙發上空出半個位子，示意我坐。舊皮的味道混著紙味，暖暖的。）

綾瀬楓：「這本是爺爺最寶貝的。你集滿了，也有資格看。要一起嗎？」

主角：「這麼珍貴的，讓我看好嗎？」

綾瀬楓：「書要有人讀才活著。放著不翻，才是糟蹋。爺爺是這麼說的。」

（我在她旁邊坐下，沙發果然陷了下去，兩個人差點撞在一起。她耳根紅了，卻沒挪開。）

主角：「……這沙發真的該修了。」

綾瀬楓：「別修。它壞成這樣，才逼得看書的人靠近一點。爺爺說不定是故意的。」

主角：「原來這店裡連壞沙發都有心機。」

綾瀬楓：「噓。小聲點——書在聽呢。」

（她低頭翻開那本舊書，早春的光從窗縫落在紙頁上。爺爺的梅花章，在扉頁角落，靜靜蓋著一枚淡淡的紅。）
```
