# 場景：歲末商店街的福引攤——轉盤前，只剩最後一張抽獎券
# 角色：立花 沙耶（たちばな さや，同班，總在班上笑得最開、其實很會照顧人、討厭輸）
# 季節：十二月（歲末，商店街掛滿福引旗）
# 出處：手寫・合世界聖經

```
（歲末的商店街，紅白條紋的福引旗在冷風裡啪嗒響。轉角那台販賣機旁邊，搭起了木頭抽獎攤。）
（攤主是米店的大叔，凍得直搓手。搖獎的六角轉盤發出嘩啦嘩啦的聲音。）

立花沙耶：「來得正好！你買東西了嗎？滿五百就有一張券。」

主角：「買了條圍巾。所以有一張。」

立花沙耶：「我攢了一整條街，八張。加你這張，九張。」

主角：「妳把整條商店街逛成了福引礦場。」

立花沙耶：「歲末就是要這樣玩嘛。走走走，一起搖。」

（她把八張券排開像一副牌，眼睛亮得跟旗子上的燈一樣。第一搖，白球——謝謝惠顧。第二搖，還是白球。）

立花沙耶：「……大叔，你這轉盤是不是壞了。」

攤主：「小姑娘，運氣的事，急不來喔。」

立花沙耶：「我最討厭聽人家說運氣。運氣是搖出來的。」

（一張接一張，全是白球。到最後只剩我那一張。她忽然安靜下來，把券推到我面前。）

立花沙耶：「你搖。我今天手氣被凍壞了。」

主角：「妳確定？這可是妳礦挖了一整條街的最後一鏟。」

立花沙耶：「所以才給你。我搖不出的，說不定你行。快啦。」

（我搖了。轉盤慢下來，滾出一顆——綠球。攤主愣了一下，敲響了桌上那口小銅鑼。）

攤主：「三獎！日和溫泉招待券，兩張！」

立花沙耶：「兩張……!」

主角：「妳看，運氣真的是搖出來的。只是搖的人不一定是妳。」

立花沙耶：「……哼，算你贏。那，其中一張，你要陪我去用掉。這不算欠人情，算……分紅。」

主角：「行。年初的溫泉，分紅照付。」

（她笑得比旗子上所有的燈都亮，把另一張券小心收進圍巾裡。）
```
