# 場景：定食屋「陽溜亭」——傍晚沒人的時段，一杯冰淇淋汽水配一個搭錯話的人
# 角色：東雲 ひかり（しののめ ひかり，鄰校高一，轉來日和町不久、有點放不開、假裝很熟這座小城）
# 季節：八月（盛夏，傍晚仍熱）
# 出處：手寫・合世界聖經

```
（陽溜亭的午市早散了，晚餐時段還沒到。店裡只有電風扇轉著，收音機播著棒球轉播，冷氣終於贏過了煎台。）
（我一個人佔了靠窗的位子，點了杯冰淇淋汽水。藍色的汽水上浮著一球香草，正是這種發呆的傍晚該有的東西。）

東雲ひかり：「不好意思……請問，這家的冰淇淋汽水，是招牌嗎？」

（隔壁桌的女生探過身來。制服不是日和高的，是鄰校的。臉有點生，笑得有點緊張。）

主角：「算是吧。老闆說配方三十年沒改過。妳沒喝過？」

東雲ひかり：「嗯……我剛搬來這附近不久。還在，呃，熟悉這座城。」

主角：「看得出來。妳剛剛研究菜單研究了很久。」

東雲ひかり：「……有那麼明顯嗎。我想假裝自己是常客。」

主角：「假裝常客的人，不會把『日替定食』唸成『日誰定食』。」

東雲ひかり：「你聽到了喔——好丟臉。」

（她整張臉紅到耳根，我沒忍住笑，把菜單翻到背面推給她。）

主角：「這面是老闆手寫的隱藏菜單。薑汁燒肉加一份可樂餅，是這裡的內行吃法。」

東雲ひかり：「內行吃法……你教我，我就真的像常客了嗎？」

主角：「至少不會再唸錯菜名。要不要先從這杯開始？」

（我招手替她也點了一杯。老闆娘端來時，兩杯汽水並排冒著小泡，藍得一模一樣。）

東雲ひかり：「哇……真的好藍。這顏色喝下去，感覺整個夏天都涼了。」

主角：「妳看，這樣講就很像本地人了。」

東雲ひかり：「是你教得好啦。……那，我可以常常來這裡問你嗎？關於這座城的事。」

主角：「可以。反正這種傍晚，我多半也在這發呆。」

東雲ひかり：「太好了。那從今天起，這杯冰淇淋汽水，就是我在日和町的第一個『常去』。」

（風扇把兩人桌上的紙巾吹得輕輕翻動，窗外的蟬聲，慢慢被傍晚的風收薄了。）
```
