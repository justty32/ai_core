# 場景：商店街的和菓子老店——玻璃櫃前，挑一顆說不清是為誰買的生菓子
# 角色：甘露寺 睦月（かんろじ むつき，同校高一，和菓子店的女兒、溫吞、對甜點極講究、對人有點迷糊）
# 季節：四月（春，櫻花剛落）
# 出處：手寫・合世界聖經

```
（商店街深處的和菓子老店，木頭櫃檯擦得發亮。玻璃櫃裡，一排排生菓子做成花的樣子，粉的、綠的、淺黃的。）
（我只是路過，卻被那顆做成櫻花的和菓子勾住了腳步。）

甘露寺睦月：「歡迎光臨……啊，你看很久了。那顆是『花筏』，這禮拜的季節生菓子。」

主角：「花筏？」

甘露寺睦月：「落在水面、被河水載著走的花瓣，就叫花筏。這個時節，霞川上到處都是。」

主角：「所以妳把河面上的花，做成了一顆點心。」

甘露寺睦月：「嗯……賣完這禮拜就沒有了。季節過了，就要換下一個。和菓子是這樣的，只留一小段時間。」

主角：「那有點可惜。」

甘露寺睦月：「所以才要現在買呀。你要一顆嗎？」

主角：「……我在猶豫。我又不太懂這個，怕糟蹋了。」

甘露寺睦月：「不會糟蹋的。好不好吃，是舌頭決定，又不是懂不懂決定。」

（她從櫃子裡取出那顆，動作輕得像捧著真的花。指尖沾了點糖粉，她也沒察覺。）

甘露寺睦月：「你手心攤開。它很軟，要用看的、用抿的，不能咬。」

主角：「這麼講究。」

甘露寺睦月：「一年只有這幾天能吃到，當然講究。……啊，我糖粉沾到你手上了，對不起。」

主角：「沒事。妳做點心的時候，是不是常常忘記自己弄了一手粉？」

甘露寺睦月：「……常常。爸爸也老這麼說我。只要一碰到菓子，別的就顧不上了。」

（她把那顆花筏包進印著店號的小紙裡，忽然多放了一顆綠色的進去。）

甘露寺睦月：「這顆是『若葉』，新葉的意思。多送你的。花會落，葉子會接著長出來——這樣，季節就不算過完了。」

主角：「……謝謝。這個道理，比點心還甜。」

甘露寺睦月：「哪有人這樣講的。快趁新鮮吃，別讓它等太久喔。」

（春末的光從店門斜進來，照著玻璃櫃裡那一整排，只開這幾天的花。）
```
