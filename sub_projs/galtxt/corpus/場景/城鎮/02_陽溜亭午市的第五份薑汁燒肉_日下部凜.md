# 場景：定食屋「陽溜亭」——午市最忙時被抓去顧店幫忙
# 角色：日下部 凜（くさかべ りん，同校高一，陽溜亭老闆的姪女，暑假來幫忙、俐落、刀子嘴）
# 季節：七月（盛夏，午市正忙）
# 出處：手寫・合世界聖經

```
（盛夏的正午，陽溜亭的門簾被熱風掀著。冷氣拚了命也壓不住煎台那頭的油香。）
（我只是進來吃頓日替定食，卻在門口被塞了一條圍裙。）

日下部凜：「你，會端盤子嗎？」

主角：「……我是客人。」

日下部凜：「客人現在最閒。三桌薑汁燒肉，我一個人手不夠。叔叔的姪女不能倒，你可以。」

主角：「這什麼歪理。」

日下部凜：「歪理能讓客人不等到翻臉，就是好理。快，那份可樂餅要涼了。」

（她把定食往我托盤上一放，動作快得像練過。汗貼在她額前，她卻連擦都沒空。）

主角：「靠窗那位老爺爺點的。冰淇淋汽水呢？」

日下部凜：「你還記得加點？行啊你。汽水在冰櫃，最下層。別搖，會噴。」

（我端著走過去，玻璃杯裡的汽水冒著小泡，藍得像假的。老爺爺笑著跟我道謝，我莫名有點得意。）

日下部凜：「別站著發呆。回來收二號桌。」

主角：「妳這人使喚起人來真順。」

日下部凜：「因為你端得比我想的穩。」

主角：「……那算誇獎？」

日下部凜：「算。我很少誇人的，聽好了。」

（午市的尖峰過去，店裡只剩風扇和收音機。她終於坐下，替自己也替我各盛了一份剩下的薑汁燒肉。）

日下部凜：「工錢我沒有。這份請你。叔叔的手藝，值了吧。」

主角：「……值。比我原本要點的還好吃。」

日下部凜：「那明天同一時間。你已經是半個陽溜亭的人了。」

主角：「等等，這是我自願的嗎——」

日下部凜：「圍裙都穿過了，來不及反悔囉。」
```
