# 場景：上課被老師點名、同桌偷偷提示
# 角色：綾瀨 千歲（同桌、鬼靈精、膽子大、表面吊兒郎當）
# 季節：初雪（十二月、上午的古文課、窗外飄著細雪）

```
（古文課上到一半，窗外開始飄今年第一場雪，我看得出了神。）
（老師的粉筆聲停了。「主角，第三句怎麼解。」——完了，我根本沒在聽。）
（我僵在原地，隔壁的綾瀨同桌用手肘頂了我一下。）

綾瀨千歲：「（氣音）『思ふ』……想念的意思，主詞是女的。」
主角：「……呃，是、是女子在思念遠方的人。」
（老師點點頭，轉身繼續寫板書。我整個人洩了氣。）

綾瀨千歲：「（壓低聲）反應這麼慢，剛剛在看雪對吧。」
主角：「……妳怎麼知道。」
綾瀨千歲：「你嘴角都翹起來了，一看就沒在念書。」

（她攤開課本擋著臉，笑得肩膀微微抖。）

主角：「謝了。差點當眾社會性死亡。」
綾瀨千歲：「小事。同桌就是拿來墊背的，互相互相。」
主角：「那下次換妳被點，我可不一定救得回來。」
綾瀨千歲：「你敢見死不救，我就把你課本角落那些塗鴉全給老師看。」

（我心頭一驚——那些畫她什麼時候看到的。）

主角：「……妳偷看我課本？」
綾瀨千歲：「同桌哪有偷不偷，都在你手邊晃。畫得還行，那隻是河鹿吧。」
主角：「……別到處說。」
綾瀨千歲：「保密可以，交換條件——下課帶我去看天台的雪。」

（她指了指窗外，雪已經在對面校舍的屋簷積了薄薄一層。）

主角：「天台明明沒鎖，妳自己去不就好了。」
綾瀨千歲：「一個人看雪多沒意思。而且我剛救了你一命耶。」
主角：「……這人情利息也太高。」
綾瀨千歲：「嫌貴？那我現在就舉手跟老師說我不會第四句，看誰先社死。」

（她作勢要抬手，我趕緊按住她的袖子。她得意地哼了一聲。）
（窗外的雪越飄越密。看樣子這一節古文，我是別想專心了。）
```
