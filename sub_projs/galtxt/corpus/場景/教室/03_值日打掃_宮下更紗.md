# 場景：值日打掃的教室
# 角色：宮下 更紗（同班、慢性子、圖書委員、愛看舊書）
# 季節：梅雨

```
（值日表把我跟宮下同學排在一起。外面下著梅雨，雨聲細細密密。）
（她掃地掃得很慢，像在數地板的紋路。）

主角：「這樣掃到天黑都掃不完。」
宮下更紗：「反正末班車還早，日和站的木頭牌子上寫，六點半才最後一班。」
宮下更紗：「急什麼呢。」

（她把畚箕遞給我，自己去擦黑板。粉筆灰在雨天的濕氣裡飄得特別慢。）

主角：「妳每天都這麼不慌不忙？」
宮下更紗：「河鹿堂的老闆說，看舊書的人走路都會變慢。」
宮下更紗：「我大概是被那家店養壞了。」

（她擦到一半停下來，看著窗外的雨。）

宮下更紗：「你看，霞川的水漲起來了。梅雨一來，河堤的草會綠得發亮。」
主角：「妳掃地掃到看風景。」
宮下更紗：「掃地是身體的事，看風景是眼睛的事，不衝突。」

（我一時無話。她說話總有種奇怪的道理。）

主角：「……那個角落還有紙屑。」
宮下更紗：「知道了，會掃到的。世界不會因為晚五分鐘就壞掉。」
主角：「但值日會被老師唸。」
宮下更紗：「那我幫你擋。反正老師也拿慢的人沒辦法。」

（她笑了一下，繼續慢慢地擦。雨還在下，教室裡只有掃帚和雨聲。）
（跟她一起值日，時間好像真的變慢了。也不算壞。）
```
