# 場景：學期換座位
# 角色：三雲 小春（同班、開朗黏人、園藝社、怕鬼）
# 季節：梅雨

```
（換座位抽籤，全班鬧哄哄。抽完號碼，三雲同學蹦到我旁邊，一臉如釋重負。）

三雲小春：「太好了，抽到你旁邊！」
主角：「……妳有必要這麼開心嗎。」
三雲小春：「有啊。上次抽到窗邊角落，那個位置對著走廊盡頭，梅雨天陰森森的，我怕。」

（她一邊說一邊把小盆栽搬過來，擺在桌角。是園藝社養的。）

主角：「妳連盆栽都跟著搬。」
三雲小春：「這是我養的三色堇，不放身邊會擔心。梅雨太濕，容易爛根。」
主角：「……妳對植物比對人上心。」
三雲小春：「才沒有，我對人也很上心的。比如現在——你午餐都吃陽溜亭的可樂餅對不對？」
主角：「妳怎麼知道。」
三雲小春：「便當盒的味道會飄過來。我鼻子很靈的。」

（她湊過來看我桌上的美術作業，下巴幾乎要擱到我肩上。）

主角：「太近了。」
三雲小春：「因為要看清楚嘛。你畫的是日和橋？」
主角：「梅雨的河，練水彩。」
三雲小春：「濕答答的，畫得跟真的一樣。我喜歡。」

（她說喜歡的時候一點都不遮掩，反而是我先移開了視線。）

三雲小春：「以後同桌，很多事都拜託你囉。特別是打雷的時候。」
主角：「打雷跟我有什麼關係。」
三雲小春：「有關係！你要假裝很鎮定的樣子給我看，我就不那麼怕了。」
主角：「……這什麼道理。」
三雲小春：「同桌的道理。」

（她笑得眼睛彎彎，把盆栽又往中間挪了挪，像是把我也一起圈進她的地盤。）
```
