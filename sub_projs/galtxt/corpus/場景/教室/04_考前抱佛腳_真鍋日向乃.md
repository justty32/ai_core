# 場景：考前抱佛腳的教室
# 角色：真鍋 日向乃（同班、樂天派、成績吊車尾、社交強）
# 季節：初雪（冬、期末考週）

```
（午休的教室成了臨時自習室。窗外飄著今年的初雪。）
（真鍋同學抱著一疊講義坐到我旁邊，表情像來借錢的。）

真鍋日向乃：「主角，救命。數學明天考，我還在第一章。」
主角：「……現在是第五章。」
真鍋日向乃：「所以才叫救命啊！」

（她把講義攤開，上面畫的塗鴉比公式還多。）

主角：「妳這節課到底在幹嘛。」
真鍋日向乃：「我在畫日和橋的雪景，很美吧？剛剛下課從走廊看到的。」
主角：「……妳完蛋了。」
真鍋日向乃：「我知道，所以我把希望寄託在你身上。」

（我嘆氣，把她的筆搶過來，圈出最可能考的三題。）

主角：「這三題背熟，至少不會零分。」
真鍋日向乃：「你人真好，比商店街發糖的大嬸還好。」
主角：「這什麼比喻。」
真鍋日向乃：「誇獎啊！那個大嬸超好的。」

（她認真了大概三分鐘，又抬起頭。）

真鍋日向乃：「考完一起去陽溜亭吃紅豆湯圓好不好？下雪天就是要吃那個。」
主角：「先把這題算對再說。」
真鍋日向乃：「算對就去喔？你說的！」
主角：「……我什麼都沒答應。」
真鍋日向乃：「嘿嘿，聽到了聽到了。」

（她重新低頭，筆尖終於在草稿紙上動了起來。窗外的雪越下越密。）
（真希望她這股勁能撐到明天早上。大概撐不到，但還是有點想幫她。）
```
