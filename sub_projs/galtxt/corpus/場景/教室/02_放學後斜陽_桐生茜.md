# 場景：放學後斜陽的教室
# 角色：桐生 茜（鄰班、直爽、田徑社、大嗓門但心細）
# 季節：盛夏

```
（值日生把桌椅排完就跑光了。我留下來擦沒擦完的窗。）
（夕陽把整間教室染成橘色，連黑板都像燒起來。這時候門被踹開——不是誇張，是真的踹。）

桐生茜：「欸！這裡有沒有一本藍色封面的筆記——啊，你！」
主角：「……妳走錯教室了，這裡是二班。」
桐生茜：「我知道啦，我借人的東西輾轉流到你們班了。」

（她風風火火地翻起最後一排的桌肚，馬尾一甩一甩的。）

主角：「藍色的？剛剛掃地掃出來一本，放講台了。」
桐生茜：「你怎麼不早說！」
主角：「妳話比我快。」

（她抱住筆記，鬆一口氣，整個人才慢下來。斜陽照著她跑到微紅的臉。）

桐生茜：「……剛從河堤練完長跑，繞霞川一圈，快熱死。」
主角：「這種天氣還跑？」
桐生茜：「文化祭有接力，不練不行。」
桐生茜：「欸，你要不要一起？男生一組還缺人。」

（我正想拒絕，她已經自顧自地在筆記上寫下我的名字。）

主角：「我沒答應。」
桐生茜：「陽溜亭的可樂餅，跑完我請。這樣呢？」
主角：「……妳很會抓人弱點。」
桐生茜：「田徑社的直覺。就這麼定了！」

（她又風一樣地衝出去，走廊迴響著她的腳步。教室重新安靜，只剩我和半扇沒擦完的窗。）
（可樂餅這種東西，怎麼就答應了呢。）
```
