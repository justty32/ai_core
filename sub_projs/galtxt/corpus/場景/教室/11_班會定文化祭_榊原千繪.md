# 場景：決定文化祭攤位的班會
# 角色：榊原 千繪（同班、學級委員、責任感過剩、其實很怕冷場）
# 季節：秋（十月、文化祭前三週）

```
（六限的班會，黑板上寫著「文化祭・本班企劃」，底下一片沉默。）
（榊原同學站在講台前，粉筆握了三分鐘，全班沒人舉手。）

榊原千繪：「所以……真的沒有人有想法嗎？鬼屋、咖啡廳、都可以。」
（一片沉默。有人假裝在看窗外的杜坂。）
榊原千繪：「主角同學，你選修美術，佈置那塊你最有發言權。」

（她的視線筆直地釘過來，我想躲都沒地方躲。）

主角：「……我只是會塗牆，不是會出主意。」
榊原千繪：「塗牆就很好了！至少你會動手，比只會嘆氣的某些人強。」
（後排傳來一陣笑，她臉紅了一下，但沒退。）

主角：「那……做個『日和町懷舊茶屋』？擺成商店街那種老木頭味。」
榊原千繪：「懷舊茶屋……可以掛河鹿堂那種暖簾，燈調暗一點。」

（她轉身在黑板上飛快地寫，粉筆敲得篤篤響，眼睛突然亮了。）

榊原千繪：「主角同學負責主視覺牆。有異議的人現在說，不然就這樣定了。」
主角：「喂，我什麼時候答應——」
榊原千繪：「你剛剛出主意的時候。開弓沒有回頭箭。」

（全班竟然難得地響起了附和聲，這下我是真的跑不掉了。）

主角：「……妳這是把冷場的鍋甩給我。」
榊原千繪：「不是甩鍋，是託付。差很多的。」
主角：「聽起來一模一樣。」
榊原千繪：「等牆畫好你就懂了。到時候第一個謝我。」

（下課鈴響，同學一哄而散，只剩她一個人擦黑板。）
（我猶豫了一下，走過去接過另一支板擦。她愣了愣，沒說話，只是嘴角動了一下。）
```
