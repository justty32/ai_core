# 場景：下課借筆記的教室
# 角色：相澤 結菜（同班、迷糊、字很漂亮、常常搞丟自己的東西）
# 季節：五月（初夏、感冒缺課後）

```
（早自習前，相澤同學抱著書包衝進來，頭髮還帶著跑上杜坂的亂。）
（她在我桌前刹住，喘得像剛從河堤追公車回來。）

相澤結菜：「主角同學！救我！古文我缺兩堂，明天要考。」
主角：「……妳感冒好啦？」
相澤結菜：「好了好了，鼻子還有點塞。筆記，拜託借我抄！」

（我翻出筆記本遞過去，她接的時候差點沒抓穩，手忙腳亂地接住。）

主角：「妳自己的呢。」
相澤結菜：「上禮拜不小心夾進圖書館還的書裡了，找不回來。」
主角：「……那本書要是被借走，妳筆記就環遊世界了。」
相澤結菜：「別說了，我已經很想哭了。」

（她翻開我的筆記，眼睛忽然亮起來，指著頁邊的小插圖。）

相澤結菜：「你連課本角落都畫小圖！這隻是……河鹿？」
主角：「無聊塗的，別看那個。」
相澤結菜：「好可愛。我抄完還你的時候，也在旁邊回畫一隻好不好？」
主角：「那是我的筆記，不是交換日記。」

（她已經拿出自己那支漂亮的鋼筆，笑得毫無誠意的乖巧。）

相澤結菜：「就一隻嘛。當作借閱費，比河鹿堂的梅花章便宜。」
主角：「……妳們是不是都拿那個章當貨幣。」
相澤結菜：「因為它可愛啊。可愛的東西就是有價值。」

（上課鈴響，她抱著我的筆記回座位，回頭補了一句。）

相澤結菜：「放學前一定還你！要是又搞丟，我把我便當分你一半。」
主角：「我寧可要筆記。」
相澤結菜：「筆記加便當，這樣總行了吧！」

（她坐下就埋頭猛抄，鋼筆刷刷地響。）
（那手字漂亮得過分，可惜配上這麼個丟三落四的主人。我莫名有點想笑。）
```
