# 場景：電扇壞掉的悶熱午後教室
# 角色：夏目 涼子（同班、毒舌、怕熱、實際上很照顧人）
# 季節：盛夏（八月、補課日、教室唯一的電扇罷工）

```
（補課日的午後，教室裡那台老電扇「喀」一聲就不轉了。）
（全班的哀嚎此起彼落。夏目同學把制服領口拉開，一臉這世界虧欠她的表情。）

夏目涼子：「這學校連一台像樣的電扇都給不起，還敢立在山丘上吹風。」
主角：「……妳這抱怨的邏輯我沒聽懂。」
夏目涼子：「熱到不用有邏輯。你臉紅成這樣還有力氣挑我語病？」

（她從書包裡摸出一把摺扇，搧了兩下，忽然轉向我搧了一陣。）

主角：「喂，那是妳的風。」
夏目涼子：「借你。你再中暑倒下去，抬你的又是我，麻煩。」
主角：「所以妳是為了自己方便。」
夏目涼子：「當然。你以為我心地好？」

（風確實涼了半分。我瞥見她自己額角也全是汗，卻搧得我比較多。）

主角：「……妳自己不搧？」
夏目涼子：「我耐熱。倒是你，早上就沒精神，水喝了沒。」
主角：「妳這關心藏得也太差。」
夏目涼子：「哪有關心，我在陳述你的沒用。」

（她起身把後門也推開，穿堂風總算勉強動了起來，帶進一點蟬聲。）

夏目涼子：「陽溜亭那個刨冰，宇治金時，下課要不要去。」
主角：「妳請？」
夏目涼子：「想得美。各付各的。我只是懶得一個人排隊。」
主角：「……真是彆扭。」
夏目涼子：「彆扭的是這鬼天氣。跟我沒關係。」

（她把摺扇往我桌上一擱，說「還你風」，自己趴回桌上裝睡。）
（明明比誰都怕熱，卻先把扇子讓出來。這種人真是不老實得可以。）
```
