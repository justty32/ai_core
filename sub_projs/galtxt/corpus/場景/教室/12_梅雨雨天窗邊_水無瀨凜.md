# 場景：梅雨天窗邊的教室
# 角色：水無瀨 凜（同班、文靜、喜歡雨聲、總帶著一本舊文庫）
# 季節：梅雨（六月、放學後遲遲不停的雨）

```
（放學鈴響過很久，雨還是沒停的意思。走廊的燈一盞盞亮起來。）
（我沒帶傘，賴在教室不想動。窗邊只剩水無瀨同學一個人。）
（她沒開燈，就著窗外灰白的光看書，指尖偶爾翻頁。）

主角：「……妳也沒帶傘？」
水無瀨凜：「帶了。只是還不想走。」
主角：「這種雨有什麼好待的。」

（她抬起頭，把書輕輕合上，指節壓在書頁間當書籤。）

水無瀨凜：「你聽。雨打在雨庇上和打在窗玻璃上，聲音不一樣。」
主角：「……有嗎。」
水無瀨凜：「有。玻璃上的比較鈍，鐵皮上的比較脆。混在一起才好聽。」

（我閉上眼真的聽了一下。原來吵成一片的雨聲，拆開來是有層次的。）

主角：「妳每次下雨都這樣坐著？」
水無瀨凜：「嗯。教室空了以後，雨聲會變大。像整棟樓只剩雨在說話。」
主角：「聽起來有點寂寞。」
水無瀨凜：「是安靜。寂寞和安靜，我一直分得很清楚。」

（她把文庫本往我這邊推了半格，封面舊得看不清書名。）

水無瀨凜：「這本寫梅雨的段落很好。要不要等雨小一點，順便看兩頁？」
主角：「……我看書會睏。」
水無瀨凜：「那就睏。反正雨還要下很久。」

（她重新翻開書，側臉映著窗外的水光，睫毛上像也沾了濕氣。）
（我拉了張椅子在她斜對面坐下，沒說話。雨聲果然，慢慢變得清楚了。）

主角：「……鐵皮那個，真的比較脆。」
水無瀨凜：「你看，聽出來了。」
```
