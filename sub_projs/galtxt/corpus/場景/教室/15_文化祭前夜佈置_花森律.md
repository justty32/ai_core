# 場景：文化祭前夜佈置教室
# 角色：花森 律（同班、美術部、手巧話少、對細節龜毛）
# 季節：秋（十月、文化祭前一晚、破例留到很晚）

```
（文化祭前夜，教室被改造到一半。桌子疊在角落，牆上貼著沒乾的顏料。）
（其他人都回家了，只剩花森同學蹲在梯子下，對著一片牆板皺眉。）

花森律：「這邊的紅，太亮了。跟你調的暖簾配不起來。」
主角：「……都幾點了，還看得出差半階顏色？」
花森律：「看得出。看不出的話，明天客人也會覺得哪裡怪，只是說不上來。」

（她把調色盤遞給我，示意我再兌一點赭色進去。）

主角：「妳們美術部都這麼龜毛？」
花森律：「不龜毛的叫塗牆，龜毛的才叫佈置。」
主角：「……剛好我就是被分來塗牆的。」
花森律：「所以我留下來陪你。塗牆的旁邊，得有個看顏色的。」

（她說得理所當然，手上的排筆卻沒停，一道一道把牆補得均勻。）

主角：「妳不累嗎。」
花森律：「累。但明天燈一亮，這面牆會替我們說話。那時候就不累了。」
主角：「說話？」
花森律：「顏色會。暖的紅配舊木頭，客人一進來就會想起商店街。你不信？」

（我後退兩步看整面牆，昏黃的走廊燈打上去，那片紅真的沉了下來，像傍晚的日和町。）

主角：「……好像，真的有點那個味道了。」
花森律：「你看。所以半階很重要。」
主角：「妳這個『你看』，跟藤宮她們一個調。」

（她難得笑了一下，很淺，像怕浪費表情。）

花森律：「因為對的時候，就是想讓人看見。」
主角：「那明天最該被看見的是妳。」
花森律：「不。是這面牆。我躲在後面就好。」

（她重新蹲回梯子下，補最後一角。窗外杜坂的樹影被風吹得沙沙響。）
（她總把自己藏在作品後面。可我覺得，願意留到這麼晚的那個人，也很值得被看見。）
```
