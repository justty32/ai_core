# 場景：轉學生自我介紹
# 角色：雪村 遙（新轉入、大城市來、看似成熟其實迷路王）
# 季節：四月（新學期）

```
（早會時間，班導領著一個沒見過的女生走進來。全班的目光唰地集中過去。）
（她站在講台上，制服的森林綠還很新，肩膀繃得有點緊。）

雪村遙：「初次見面。我叫雪村遙，從東京轉來的。請多指教。」
班導：「雪村同學家搬到日和町，以後就是我們班的一員了。」

（教室裡響起零星掌聲。有人小聲議論「東京欸」。）

雪村遙：「……我對這裡還不太熟。昨天想去車站，結果走到河堤去了。」

（全班笑出聲。她臉一紅，卻也跟著笑了，緊繃的肩膀鬆下來一點。）

班導：「日和站的末班車很早，雪村同學要記得。座位就坐主角旁邊那個空位吧。」

（她走過來，在我旁邊坐下，把書包放好，動作一絲不苟。）

雪村遙：「……以後麻煩你了。這裡的路我真的會迷。」
主角：「妳連從車站到學校都會迷？就一條杜坂爬上來而已。」
雪村遙：「那個坡好陡，我今早爬到懷疑人生。」
主角：「習慣就好，全校都是這樣練出來的。」
雪村遙：「聽起來像某種修行。」

（她說話比想像中好接。原來成熟只是外殼。）

主角：「放學我可以帶妳認一下路。商店街、書店、車站，走一圈就記得了。」
雪村遙：「真的可以嗎？我怕又走到河堤。」
主角：「走到河堤也不錯，霞川傍晚很好看。」
雪村遙：「……那我更想迷路了。」

（她偏過頭笑，窗外的櫻花正好落下一片。日和町又多了一個會慢慢喜歡上這裡的人吧。）
```
