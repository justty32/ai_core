# 場景：清晨無人的教室
# 角色：鷺澤 詩織（同班、晨型人、話少、擅長素描）
# 季節：四月（櫻花剛開）

```
（早到得離譜。杜坂爬上來時霧還沒散，走廊冷得像沒睡醒。）
（推開教室門，我以為裡面沒人。）
（靠窗第二排，鷺澤同學正對著窗外畫著什麼，鉛筆的沙沙聲在空教室裡特別大。）

主角：「……早。妳都這個時間到？」
鷺澤詩織：「嗯。人少的時候光線最好。」
鷺澤詩織：「你看，櫻並木那排從這裡剛好能看到河堤。」

（我順著她的視線望出去。霞川的水面把櫻花的粉色揉碎了浮在上面。）

主角：「畫這個要交嗎？」
鷺澤詩織：「不交。畫得出來的東西，我才留得住。」
鷺澤詩織：「等花全開，這個角度就被葉子擋住了。只有現在。」

（她沒抬頭，鉛筆繼續動。我不知道該站著還是坐下。）

主角：「……那我坐這，不吵妳。」
鷺澤詩織：「你呼吸就好，別評論。」

（我差點笑出來。原來話少的人嘴也挺硬。）

主角：「畫完給我看一眼。」
鷺澤詩織：「看一眼要付錢的，河鹿堂那種梅花章一個。」
主角：「……妳去書店蓋章還蓋上癮了。」
鷺澤詩織：「一天一個。集滿一頁，換一句真話。」

（窗外的櫻花無聲地落。教室慢慢有了第二個、第三個人的腳步聲。）
（我忽然有點捨不得這段只有沙沙聲的安靜。）
```
