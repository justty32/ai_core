# 場景：忘帶課本，併桌共讀
# 角色：東雲 澪（同班、冷面、其實怕生、書法社）
# 季節：秋（文化祭前）

```
（第三節古文課，我翻遍書包才想起課本忘在家。冷汗直流。）
（老師掃視全班找共讀對象時，指向了我隔壁——東雲同學，全班公認最難搭話的人。）

主角：「……那個，抱歉，借我看一下。」
東雲澪：「桌子往這邊併。別碰到我的筆袋。」
主角：「妳筆袋裡是什麼危險物品嗎。」
東雲澪：「毛筆。文化祭書法社要展，弄髒了我跟你沒完。」

（她把課本推到兩桌中間，位置算得很精準，一人一半，公平得像用尺量過。）

主角：「妳字很好看。剛剛偷瞄到妳筆記。」
東雲澪：「……別偷瞄別人的東西。」
主角：「這也算偷瞄？」
東雲澪：「算。」

（她耳朵有點紅。原來冷面下面也不是鐵板。）

主角：「文化祭妳們社要寫什麼？」
東雲澪：「還沒定。老師說寫跟日和町有關的。」
主角：「那寫『霞川』啊，兩個字都好看。」
東雲澪：「……我本來就打算寫那個。」
主角：「看吧，我們挺合的。」
東雲澪：「別自作多情。翻頁，跟不上了。」

（她低頭遮住表情，手指卻幫我把頁碼按在正確的地方。）

主角：「謝了。下次不會忘課本了。」
東雲澪：「最好是。桌子併回去。」

（下課鈴響，她飛快地把桌子推開，像要跟我劃清界線。可課本，是她主動先合起來遞回我這邊的。）
```
