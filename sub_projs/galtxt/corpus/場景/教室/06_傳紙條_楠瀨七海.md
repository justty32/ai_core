# 場景：上課傳紙條
# 角色：楠瀨 七海（同班、鬼靈精、廣播社、坐我後面）
# 季節：盛夏

```
（悶熱的下午，數學老師的聲音像催眠曲。後背忽然被鉛筆戳了一下。）
（一張折成小方塊的紙條從肩膀後面遞過來。是坐後面的楠瀨。）

（紙條：「無聊死了。你在畫什麼？我全看到了。」）

（我在課本邊角塗鴉被抓包。回傳：「河堤的貓。別告狀。」）

（紙條：「畫得像抹布。放學帶我去看真的貓，就不告狀。」）

（我回頭瞪她。她比了個「噓」，眼睛笑成月牙。）

（回傳：「櫻並木下面那隻很兇，會抓人。」）

（紙條：「兇的才有意思。你怕貓？堂堂美術選修？」）

（回傳：「我怕的是被老師沒收紙條。妳廣播社的膽子借我一點。」）

（紙條：「膽子不外借。但可以帶你去河鹿堂，老闆養了隻超肥的店貓，不抓人。」）

主角：（小聲）「……妳到底想去看貓還是逛書店。」
楠瀨七海：（更小聲）「都要。順便蓋梅花章。」

（老師忽然轉過身，教室瞬間死寂。）

數學老師：「後面兩位，在傳什麼？」
楠瀨七海：「報告老師，在討論這題怎麼解！」
數學老師：「……那你上來解。」

（她被叫上台，臨走前把紙條塞我手裡，寫著：「放學校門口等我。逃不掉。」）
（我把紙條攥進口袋。這下午，好像沒那麼悶了。）
```
