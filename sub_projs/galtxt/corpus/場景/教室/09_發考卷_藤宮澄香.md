# 場景：發回考卷的教室
# 角色：藤宮 澄香（同班、成績頂尖、話直、對分數異常執著）
# 季節：五月（新綠、第一次段考後）

```
（老師抱著一疊考卷進來，全班的空氣瞬間凝固了半度。）
（我盯著桌角的木紋，假裝那樣就能拖延命運。）
（發到我這排時，前座的藤宮同學轉過身，先把她自己那張攤在我面前。）

藤宮澄香：「九十六。扣的四分是我不服氣的。」
主角：「……妳給我看這個做什麼。」
藤宮澄香：「參照組。你看完自己的，才知道差距是絕望還是有救。」

（我認命翻開自己的。六十一。剛好踩在及格線上，像走鋼索的人。）

主角：「有救的那種……大概。」
藤宮澄香：「給我看第三大題。」
主角：「喂，那是我的隱私——」

（她已經抽走了，眼睛掃過去的速度快得不像在看考卷。）

藤宮澄香：「觀念沒錯，計算兩次符號寫反。這不是笨，是急。」
主角：「聽起來還是在罵我。」
藤宮澄香：「陳述事實。你要是笨我懶得看第二眼。」

（她把考卷推回來，指尖在那兩處錯誤上點了點。）

藤宮澄香：「下次段考，你坐我後面剛好。錯的地方我用紅筆敲你桌子。」
主角：「那不成敲木魚了嗎。」
藤宮澄香：「敲到你開竅為止。河鹿堂的木魚都沒我勤。」

（我忍不住笑，她卻一臉正經，好像剛才說了什麼哲理。）

主角：「……為什麼特地管我。」
藤宮澄香：「因為你差的那幾分，是懶，不是不會。懶最可惜。」
主角：「這算誇獎嗎？」
藤宮澄香：「算警告。及格線很擠，別老站那。」

（她轉回去了，背影挺得筆直。）
（我低頭看那個紅圈住的六十一，忽然覺得下次好像不該只考這樣。）
```
