# 槽填充模板

> 把場景的**骨架**固化成「模板＋槽（slot）＋候選值表」。
> 填空即可組句——**可以完全不經 LLM**（純確定性），也可以把模板丟給 LLM 當骨架、只讓它填留白。
> 這是「部分求值」最直接的體現：確定的骨架先算掉，只剩真正需要創意的填空。

## 槽的記法

- `【槽名】` ＝ 可替換位置；候選值見下方各槽詞表。
- `〔可選片段〕` ＝ 可有可無的修飾。
- `{A｜B｜C}` ＝ 就地擇一。
- 模板產出的是**旁白／敘述**與**台詞鉤子**；真正的對話內容仍建議留給 LLM 依 [01](01_角色類型語言表.md)[05](05_情緒語域對照.md) 填。

## 二、開場模板（建立舞台）

**T1 — 定場白（最通用）**
```
（【時段】的【地點】，【感官細節·光】。）
〔（【感官細節·聲】。）〕
（【角色】正在【動作】。）
主角：「……。」
```
> 例：（傍晚的教室。夕陽把課桌椅拉出長長的影子。）（窗外傳來運動社團隱約的呼喊聲。）（遠藤葵正低頭抄著筆記。）主角：「遠藤，還不走嗎？」

**T2 — 偶遇（單獨相處觸發）**
```
（主角來到【地點】，本以為沒人，卻看見【角色】〔一個人〕在【動作】。）
【角色】：「啊，{你也／}……。」
```

**T3 — 季節入場（時間留痕）**
```
（【季節·標誌物】。）主角走在【地點】，〔想起／注意到〕【季節】{已經／才剛}來了。
```

## 三、互動模板（推進關係，全部須過 [03](03_禁語與OOC護欄.md)）

**T4 — 遞物／給予（含蓄好感）**
```
（【角色】把【物】{遞／推}到主角面前，〔別過頭／低著頭〕。）
【角色】：「這個給你。……〔遮掩句：才不是特地／剛好多的／順路買的〕。」
```
> 物候選：一本書、多買的可樂餅、一顆汽水糖、整理好的筆記、一條手帕。

**T5 — 共傘／擋風／讓座（貼近但不油）**
```
（【天氣阻礙：雨／冷風/人潮】。）
【角色】：「你〔沒帶傘／會冷〕吧。要不要……{一起／站過來一點}？」
（兩人{在傘下／並肩}，之間留著一點恰好的距離。）
```
> **護欄**：距離「恰好」，**不寫身體接觸的特寫**。

**T6 — 鬥嘴／彆扭（傲嬌互動）**
```
主角：「【調侃】。」
【角色】：「哼，才不是那樣。……〔小聲補一句真心話〕。」
```

**T7 — 交心（關係升階觸發，見 [02](02_敬語與稱謂階.md)）**
```
（【時段·安靜氛圍】。）
【角色】：「……其實我，〔一直／有點〕【心事】。」
（她說完，看向【景物】，沒再說話。主角也沒催。）
```

## 四、收尾模板（留白與餘韻）

**T8 — 散場·各自回家**
```
主角：「那，〔明天見／路上小心〕。」
【角色】：「嗯。……〔一句半含蓄的期待：明天也／那個…〕。」
（【時段】的【地點】，【感官細節】。主角{目送／轉身走上杜坂}。）
```

**T9 — 定格·景收**
```
（沒有人再說話。【感官細節·光】〔一點一點暗下去／落在兩人之間〕。）
（【季節·標誌物】，像是把這個〔傍晚／午後〕輕輕收了起來。）
```

**T10 — 鉤子·下次**
```
【角色】：「下次……〔一起去河鹿堂／再陪我跑杜坂／還要不要來〕？」
主角：「……好啊。」
```

## 五、組合約束（避免亂填）

模板不是隨機填，槽之間有相容性。命中約束才合世界：

| 約束 | 說明 |
|------|------|
| 【地點】↔【動作】 | 河鹿堂配「挑書」不配「跑步」；天台配「看霞川」不配「買汽水」。 |
| 【季節】↔【地點/物】 | 初雪配圍巾／暖桌／杜坂打滑；盛夏配汽水／舊電風扇／圖書室。 |
| 【角色·類型】↔【台詞語氣】 | 傲嬌填 T6，害羞填 T5，文靜填 T7——反配會怪。 |
| 【時段】↔【光/聲】 | 清晨配晨光＋無人安靜；傍晚配斜陽＋社團餘音。 |
| 稱呼 ↔ 關係階 | 模板裡的稱呼字面由 [02](02_敬語與稱謂階.md) 當前階決定，不自由填。 |

## 六、產出流程（給 pipeline）

```
1. 選骨架：季節 + 地點 + 時段 + 角色(+類型)         ← 純選擇，確定性
2. 套約束表過濾不相容組合                           ← 確定性
3. 選模板 T1..T10，填【感官/動作/物】等靜態槽         ← 確定性（可查表）
4. 台詞留白 → 交 LLM，依 01 類型語言 + 05 語域 生成    ← 唯一需創意處
5. 全段過 03 護欄 + 02 稱謂校驗                      ← 確定性檢核
```

> **固化收益**：一個場景骨架、感官鋪陳、頭尾套路——這些每次都差不多的部分**全部查表算掉**，LLM 只負責那幾句真正要有靈魂的對白。快、穩、且不漂。
