# 情緒 → 語域對照

> 把「好感度階 × 情緒」映射到具體**語域**（語氣／用詞／肢體旁白）的對照表。
> 用途：依當前情緒選詞，讓**台詞、稱呼、旁白三者不脫節**——這是含蓄系最容易漏的一致性。
> 情緒是**輸入**，語域是**輸出**；階（[02](02_敬語與稱謂階.md)）決定同一情緒能外露到什麼程度。

## 核心原則

- **同一種情緒，關係越淺越壓抑、越深越敢露**。害羞在階 1 是「別開視線」，在階 4 是「小聲承認」。
- **台詞和旁白要同向**：嘴上說「沒事」時，旁白要洩底（絞衣角／聲音小）——**言不由衷正是含蓄的甜點**，但要靠旁白讓玩家讀到。
- **不油膩總則覆蓋一切**：任何情緒的肢體外露都止於克制（見 [03](03_禁語與OOC護欄.md) A 類）。

---

## 一、階 × 外露度（總調節旋鈕）

| 階 | 外露度 | 情緒怎麼呈現 |
|----|--------|--------------|
| 0–1（生疏） | 極壓抑 | 靠旁白微表情、語域客氣、話短。幾乎不主動說情緒詞。 |
| 2（熟） | 半露 | 語氣鬆，會用軟化詞，情緒從「動作」漏出來。 |
| 3（親近） | 露一半 | 敢說半句真心話（欲言又止），旁白動作明顯些。 |
| 4+（很近） | 敢露 | 會小聲直說（但仍不濫情），沉默也有重量。 |

> 這張表是**乘數**：下面每個情緒的表現，都要再乘上當前階的外露度。
>
> **豁免條款**（跨定點驗證確立，出處：[陽溜亭驗證報告](定點_陽溜亭_初雪傍晚/01_驗證報告.md) 問題 7）：乘數的作用域＝**真心側外露**。職業性／表演性的外放（「營業聲」：店員的招呼元氣、值日生的宣達腔）**不吃壓抑**——那是制服不是情緒，階 1 也可以滿場營業元氣，不算 OOC。隨之而來的檢核判準：**元氣型的 OOC 不是「太外放」，是「外放得不分場合」**（真心側該壓抑的時刻仍用營業聲＝穿幫；反過來，營業聲滑回素的聲音＝升溫訊號）。

## 二、情緒 → 語域對照表

### 平靜／安心
| 面向 | 表現 |
|------|------|
| 語氣 | 慢、句尾軟（呢／啊）、留白多 |
| 用詞 | 「這樣就好」「不錯呢」「嗯」 |
| 旁白 | 微笑、看景（河面／窗外）、動作放輕、肩膀鬆下來 |
| NG | 過度傾訴、長篇獨白 |

### 雀躍／期待
| 面向 | 表現 |
|------|------|
| 語氣 | 語速稍快、句尾上揚、感嘆多一點 |
| 用詞 | 「欸，真的？」「那…那我們」「好期待」（依型調濃淡） |
| 旁白 | 眼睛亮、身體前傾一點、腳步輕、忍不住的小動作 |
| NG | 蹦跳尖叫（過），賣萌符號（油） |

### 害羞
| 面向 | 表現 |
|------|------|
| 語氣 | 音量小、句子說一半、停頓多 |
| 用詞 | 「那個…」「沒、沒什麼」「別看我」 |
| 旁白 | 別開視線、耳根紅、絞衣角、把臉藏進圍巾、低頭 |
| NG | 臉紅冒煙、「////」、跺腳（皆油，見 03） |

### 彆扭／傲嬌（不坦率）
| 面向 | 表現 |
|------|------|
| 語氣 | 先強後弱：否認句重、補救句輕 |
| 用詞 | 「才不是」「隨便你」「哼」＋（小聲）真心話 |
| 旁白 | 別過頭但身體沒走、嘴硬手卻幫忙、偷瞄 |
| NG | 真的惡毒、翻臉；動作與台詞同向（就沒反差了） |

### 落寞／心事
| 面向 | 表現 |
|------|------|
| 語氣 | 更慢、更輕、句尾往下沉、沉默變長 |
| 用詞 | 「沒事」「只是…有點」「你不用在意」 |
| 旁白 | 看遠處、指尖停在書頁、風吹起髮、話到一半停住 |
| NG | 哭天搶地、把苦楚說滿；讓「留白」承重 |

### 感激／被在意到
| 面向 | 表現 |
|------|------|
| 語氣 | 輕、真、稍慢，客氣體回一點點 |
| 用詞 | 「謝謝你」「…幫了我很多」「其實我一直」 |
| 旁白 | 抬頭對視一秒又移開、輕輕點頭、握緊手裡的東西 |
| NG | 感激涕零、下跪式誇張 |

### 逗弄／打趣（大姐姐、元氣）
| 面向 | 表現 |
|------|------|
| 語氣 | 上揚、帶笑、故意拖長 |
| 用詞 | 「呵呵」「真拿你沒辦法」「你臉紅了喔？」（點到即止） |
| 旁白 | 湊近半步又退開、側頭笑、手背掩笑 |
| NG | 越界肢體（勾肩摟腰＝油）、逗到對方難堪 |

### 擔心／關切
| 面向 | 表現 |
|------|------|
| 語氣 | 放輕、放慢、多一個確認句 |
| 用詞 | 「你還好吧」「別勉強」「有沒有好一點」 |
| 旁白 | 遞東西（圍巾／筆記／傘）、伸手又收回、皺眉看你 |
| NG | 母親式嘮叨長篇、過度肢體照料 |

## 三、言／行反差矩陣（含蓄系甜點的公式）

含蓄＝**嘴上一套、旁白一套**，讓玩家自己讀出真心。常用組合：

| 台詞（表面） | 旁白（真心） | 讀出的情緒 |
|--------------|--------------|------------|
| 「才不是特地給你的」 | 卻早就多買了一份 | 傲嬌的好感 |
| 「我沒事」 | 聲音很輕，握緊了書 | 落寞、逞強 |
| 「隨便你要不要」 | 卻已經往你這邊挪了半步 | 在意、口是心非 |
| 「明天…也會來嗎」 | 說完立刻補「隨口問問啦」 | 期待、害羞 |
| 「你先走吧」 | 目光卻一直沒離開 | 捨不得 |

> **規則**：反差是**旁白幫台詞補真相**，不是台詞自己前後矛盾。旁白負責「洩底」，且洩得克制——一個小動作就夠。

## 四、給 pipeline 的選詞流程

```
輸入：角色類型(01) + 當前階(02) + 目標情緒
1. 查「階 × 外露度」得壓抑等級
2. 查「情緒 → 語域」得語氣/用詞/旁白候選
3. 乘上壓抑等級（越淺越砍外露）
4. 若要含蓄甜點 → 套「言/行反差矩陣」：台詞取表面、旁白取真心
5. 出稿 → 過 03 護欄（砍掉任何越界外露）
```

## 五、一致性檢核點（拿來驗生成稿）

- 台詞語域 與 當前階 是否相符？（階 1 卻用親暱體 → 不對）
- 旁白外露度 與 階 是否相符？（生疏卻大動作肢體 → 不對）
- 台詞與旁白是否**同向或刻意反差**？（無意義的矛盾 → 打回）
- 情緒外露 是否踩到 03 的油膩線？（踩到 → 砍）
