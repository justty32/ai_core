# 場景開場／收尾套路句庫

> 場景的**頭**和**尾**最容易「每次都差不多」——正因如此最該固化。
> 這裡是**可直接抽用的成句／半成句庫**，依季節與地點分類。抽一句就能起場、收場，把 LLM 省下來給中段對白。
> 全部已對齊世界聖經與含蓄基調；抽用後仍須過 [03](03_禁語與OOC護欄.md)。

## 用法

- 句子分三種標記：
  - 【成】＝可直接用的完整旁白／敘述句。
  - 【半】＝留一個空（用 ___ 標），填地點／人／物。
  - 【鉤】＝台詞句，收尾留下下次的引子。
- 開場句負責**定場（光/聲/季節）＋引出角色**；收尾句負責**留白＋餘韻**。
- **忌重複**：同一批語料別連用同一句；把它當池子輪抽。

## 一、開場句庫（定場）

### 通用·時段光線
- 【成】清晨的教室，陽光穿過靠窗的玻璃，照亮空氣裡細小的灰塵。
- 【成】放學後的教室，夕陽把課桌椅拉出長長的影子。
- 【成】黃昏的河堤，水面被染成一片溫暖的橙色。
- 【成】入夜的車站，木造月台亮起暖黃的燈，遠處傳來電車的聲音。
- 【半】___ 的天台，風很大，鐵門被吹得輕輕晃了一下。
- 【成】午後的圖書室悶悶的，只有那台舊電風扇有一下沒一下地轉著。

### 季節·開場
- 【成】四月的杜坂，櫻花已經開始散了，一路鋪成淺淺的花毯。
- 【成】梅雨的午後，雨聲把整座校舍都包了起來。
- 【成】盛夏的蟬鳴吵得人發睏，汽水瓶上結了一層水珠。
- 【成】文化祭前一週的社辦，到處是瓦楞紙和沒乾的顏料。
- 【成】初雪落下的傍晚，杜坂的坡道泛著薄薄的白。

### 引出角色·偶遇
- 【半】主角推開門，本以為沒人，卻看見 ___ 一個人待在那裡。
- 【半】拐過商店街的轉角，正好遇上 ___。
- 【半】河堤的長椅上，___ 坐在那兒，沒注意到有人來。
- 【成】「啊，」對方先抬起頭，「……你也來了。」
- 【半】___ 正低頭做著自己的事，木地板隨著主角的腳步輕輕咯吱響。

### 起手·主角開口
- 【成】主角：「……早。」
- 【半】主角：「___，還不走嗎？」
- 【成】主角：「今天也一個人？」
- 【成】主角：「這裡風大，別站太邊邊。」

## 二、收尾句庫（留白餘韻）

### 散場·道別
- 【鉤】主角：「那，明天見。」／對方：「嗯，……明天見。」
- 【鉤】「路上小心。杜坂那段，慢慢走。」
- 【半】主角目送 ___ 的身影消失在坡道盡頭，才轉身往回走。
- 【成】兩人在車站前分開，末班車的燈在遠處一點一點靠近。

### 定格·景收
- 【成】沒有人再說話。夕陽一點一點沉下去，把兩人的影子拉得很長。
- 【成】風把最後幾片櫻花吹過河面，像是把這個午後輕輕收了起來。
- 【成】雪還在下，安靜得聽得見自己的呼吸。
- 【成】教室重新安靜下來，只剩窗外遠處運動社團的餘音。
- 【半】___ 沒有回答，只是看著窗外，嘴角有一點很淡很淡的笑。

### 鉤子·下次
- 【鉤】對方：「下次……要不要再一起走這條路？」／主角：「……好啊。」
- 【鉤】「河鹿堂說週末會進新書。你要一起去嗎？」
- 【鉤】對方：「那個——沒事。明天也會來吧？」
- 【鉤】主角：「下次換你請客。」／對方：「哼，看你表現。」

### 含蓄·未說出口
- 【成】想說的話最後沒說出口，就這樣被夜色和電車聲一起帶走了。
- 【成】有些話留到明天也不遲——至少主角是這麼想的。
- 【半】___ 揮了揮手，轉身時的表情，主角沒看清。

## 三、按場景原型的頭尾配對（快取組合）

依 [04](04_槽填充模板.md) 的常見原型，預組好的頭尾，直接抽整組：

| 場景原型 | 開場抽 | 收尾抽 |
|----------|--------|--------|
| 清晨值日·教室 | 通用晨光 ＋ 起手「早」 | 景收（教室安靜）＋ 淡淡的笑 |
| 放學繞路·河堤 | 黃昏河堤 ＋ 偶遇長椅 | 定格景收 ＋ 鉤子「再一起走」 |
| 共傘·梅雨玄關 | 梅雨午後 ＋ 對方先抬頭 | 道別 ＋ 未說出口 |
| 天台午休 | 天台風大 ＋ 「這裡風大」 | 景收 ＋ 含蓄未說 |
| 文化祭趕工·社辦 | 社辦瓦楞紙 ＋ 低頭做事 | 鉤子「明天也會來吧」 |
| 初雪·杜坂 | 初雪傍晚 ＋ 偶遇 | 目送坡道 ＋ 鉤子下次 |
| 末班車·月台 | 入夜車站 ＋ 「一個人？」 | 末班車靠近 ＋ 未說出口 |
| 河鹿堂邂逅 | （書店可自寫）＋ 偶遇 | 鉤子「週末進新書」 |

## 四、維護說明

- 這是**活句庫**：新場景寫出好的頭尾句，回頭補進對應分類。
- 抽用後**務必微調**避免逐字重複（改一個景物、換一個語氣詞即可）——固化是省骨架，不是複製貼上。
- 任何抽用句仍要過 [03](03_禁語與OOC護欄.md) 與 [02](02_敬語與稱謂階.md) 稱謂階校驗（句庫裡的稱呼是佔位，實際依當前階替換）。
