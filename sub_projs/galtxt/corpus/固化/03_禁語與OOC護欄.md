# 禁語與 OOC 護欄

> 這個**不油膩、含蓄、模糊現代日本**的世界**不會出現**的詞與橋段。
> 固化成**黑名單**，當**確定性護欄**：任何生成（套模板或自由發揮）都先過這關，命中即攔／打回重寫。
> 護欄優先級最高——它覆蓋其他所有規則。

## 怎麼用

- **生成後過濾**：掃台詞＋旁白，命中禁詞或禁橋段 → 標記 → 打回重寫或人工審。
- **生成前預防**：把「總則」塞進 prompt 當硬約束。
- 分三類：**A 油膩／擦邊**（最嚴，破基調）、**B 現代穿幫**（破「模糊年代」）、**C 語體出戲**（破輕小說口吻）。
- 禁詞是**訊號不是絕對字串比對**：重點是**意圖**。同義改寫仍越界一樣攔（用語義判，不只字面）。

---

## A 類：油膩／擦邊（最高優先，破「不油膩」基調）

### A-1 禁詞（用語表達）

| 禁 | 為什麼 | 改往哪走 |
|----|--------|----------|
| 身體部位的性化描寫（胸／腿／肌膚的挑逗性特寫） | 直接油 | 完全不寫，或只寫「側臉／髮梢／指尖」等含蓄處 |
| 「心跳漏了一拍」「臉紅到脖子」「小鹿亂撞」等濫情套語 | 廉價、油 | 用留白：停頓、移開視線、話說一半 |
| 「不小心跌進懷裡」「摸到了不該摸的」等事故式親密 | 後宮套路 | 刪。真要接觸，止於「肩並肩」「遞東西指尖碰到」 |
| 「老公／老婆／親愛的／寶貝」 | 過親、油 | 依 [02](02_敬語與稱謂階.md) 稱謂階 |
| 「//////」「///」「❤」「(*/ω＼*)」等賣萌符號 | 出戲、油 | 用文字寫克制的害羞 |
| 露骨情話（「我的心裡只有你」式直球轟炸） | 太滿、不含蓄 | 側寫、借景、欲言又止 |
| 「呀」「討厭啦～（嗲）」誇張嬌嗔 | 油 | 極輕的「真是的」「別鬧了」 |

### A-2 禁橋段

- **走光／換衣間／浴室／偷看**任何變體。
- **壁咚、公主抱、突然拉近臉、耳邊低語**等強勢肢體壓迫。
- **醉酒失態、藥物、被灌醉**。
- **後宮式**：多女角同時爭奪、吃醋大戰、「大家都喜歡主角」。
- **意外性接觸**當笑點（跌倒、絆倒、風吹裙）。
- **主角油嘴撩妹**：主角應**含蓄體貼**，不是把妹達人。

> **判準**：一句話若「拿掉會讓場景更乾淨、更像真實高中生的含蓄」，那它多半就是油。**寧可少，不可膩。**

## B 類：現代穿幫（破「模糊的現代日本」）

「模糊」＝有現代生活感，但**不點名真實品牌／時事／科技噱頭**，避免把時代釘死或跳戲。

| 禁 | 說明 | 改用 |
|----|------|------|
| 真實品牌名 | LINE、IG、TikTok、iPhone、星巴克、麥當勞、7-11… | 「手機」「訊息」「傳個訊息」「便利商店」「連鎖咖啡店」 |
| 真實地名／國名時事 | 東京鐵塔、澀谷、疫情、選舉、某年某事件 | 只用架空的「日和町」與泛稱 |
| 過度科技噱頭 | AI 助理、VR、直播帶貨、迷因梗 | 這個悠緩小城不玩這些；頂多一支普通手機 |
| 網路流行語／迷因 | yyds、絕絕子、栓Q、破防、awsl | 一律不用（含 C 類） |
| 出戲的西式元素 | 聖誕交換禮物大轟趴、萬聖裝… | 用日系季節事件（文化祭、初雪、祭典） |

> **手機的分寸**：可以有手機、可以傳訊息、可以查末班車時刻——但**畫面永遠在人與景**，不在螢幕裡的 App。手機是道具，不是舞台。

## C 類：語體出戲（破輕小說含蓄口吻）

| 禁 | 為什麼 | 改用 |
|----|--------|------|
| 網路化／簡體網感詞：超級無敵、xswl、笑死、絕了、破防、擺爛 | 跳出輕小說語域 | 「很／非常」「真好笑」「有點沮喪」 |
| 髒話、粗口、地域方言口頭禪 | 破乾淨青春感 | 最重到「笨蛋」「真是的」為止 |
| 中二台詞、戰鬥／異能腔 | 這是日常系，不是戰鬥番 | 貼地的生活語言 |
| 說明書腔／百科腔的旁白 | 破敘事沉浸 | 用感官細節寫景（光、風、聲、味） |
| 現代辦公／商業黑話：賦能、對齊、閉環、KPI | 徹底出戲 | 學生生活用語 |
| 過度書面成語堆砌 | 假、不像高中生說話 | 口語、短句、留白 |

## D 類：世界設定硬約束（別寫錯設定）

這些不是「不能出現」，而是「**出現就要對**」——寫錯即穿幫：

- 學校在**山丘上**，通學要爬**杜坂**（陡坡）。別寫「走平路到校」。
- 天台**沒上鎖**（半默許祕密基地）。別寫「翻牆上天台／天台鎖著」。
- 車站是**老木造月台，末班車很早**。別寫「深夜還有車／新式電子閘門」。
- 制服＝**森林綠**。別寫別的顏色。
- 文化祭在**秋**。別把它挪到春夏。
- 河＝**霞川**、橋＝**日和橋**、書店＝**河鹿堂**（蓋**梅花章**）、定食屋＝**陽溜亭**。名字別亂改。
- 主角＝**男・高一**。別寫成女、別寫成高二以上（除非時間線推進且交代）。

## 護欄執行順序（給 pipeline）

```
生成候選句/段
  → [A 油膩檢查]  命中 → 打回（最高優先）
  → [B 穿幫檢查]  命中 → 替換泛稱 或 打回
  → [C 語體檢查]  命中 → 改寫語域
  → [D 設定檢查]  命中 → 修正設定
  → 全過 → 交下一關（稱謂階校驗 02 / 語域對照 05）
```

> 護欄的價值在於**確定性**：這些判斷**不需要每次問 LLM「這樣油不油」**，能規則化的先規則化，把 LLM 的判斷力留給真正模糊的邊界案例。
