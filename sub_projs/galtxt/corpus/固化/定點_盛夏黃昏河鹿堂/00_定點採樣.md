# 定點採樣——盛夏黃昏・河鹿堂・柏木秋穗

> 固化階梯的**下一步實驗**：把 [04 槽填充模板](../04_槽填充模板.md) 的所有槽**釘死在一個座標點**上，然後採樣「這個定點上可能長出哪些對話」。
> 目的＝量測**殘餘創意空間**：槽全固定之後，對話還能沿哪幾條軸變、每軸有幾個候選、真正不可固化的核心剩多少。
> 結論先講：**十一個槽釘死後，殘餘自由度只剩四條軸，且每條軸自己也長得像可枚舉的表**——見文末〈觀察〉。

## 三、採樣

### 樣本 1 〔A1 書・B1 她讓書・C2 摩挲頁角・T10 鉤子〕

```
【盛夏黃昏・河鹿堂】

（推開卡卡的木門，門鈴響了一聲。外頭的蟬鳴被關在門外，只剩舊紙的味道，和二樓隱約的相撲轉播。）
（本以為店裡沒人，二手書的竹籃前卻蹲著一個熟悉的黑短髮。）

主角：「柏木？圖書室不去，跑來這裡蹲著。」
秋穗：「……圖書室的電風扇，壞第三天了。」
（她沒回頭，指尖翻過一本黃褐色的文庫本。）
秋穗：「這裡比較涼。……還很安靜。本來是。」
主角：「喔，那我閉嘴。」
（說是這麼說，我還是在旁邊的矮台前停下來，隨手抽了一本。）
主角：「……有栖川有栖。你上次唸過這名字，害我記住了。」
（翻頁的手停了一下。）
秋穗：「……記這個做什麼。考試又不會考。」
主角：「就是不會考才記得住啊。」
秋穗：「……什麼歪理。」
（她低下頭，把左側的頭髮塞到耳後——那是她要認真看過來之前的小動作。）
秋穗：「那本是短篇集。……給第一次讀的人，剛剛好。」
主角：「哦？那我買了。」
秋穗：「沒什麼特別的。只是剛好而已。」
（指尖在自己那本書的頁角，輕輕摩挲了一下。爺爺在櫃台後面醒過來，笑瞇瞇地往扉頁蓋上一枚小小的梅花章。）
主角：「那，明天見。」
秋穗：「嗯。……佐伯。」
（我回頭。她已經又低下頭去了。）
秋穗：「……讀完再來。爺爺快進新書了。」
```

### 樣本 2 〔A2 舊車票・B3 無遞物・C1 往懷裡挪・T9 定格〕

```
【盛夏黃昏・河鹿堂】

（門鈴。橘紅色的餘暉從街那頭斜斜照進店裡，把書架的影子拉得很長。）
（柏木秋穗蹲在打折區的竹籃前，手裡的文庫本翻在扉頁那一頁，很久沒有動。）

主角：「……柏木？書拿反了嗎。」
秋穗：「沒有。」
（她把書豎起來一點。扉頁角落一枚褪色的梅花章，章的下面，夾著一張泛黃的硬紙片。）
秋穗：「……車票。霞川線的。很舊很舊的年份。」
主角：「哇，比爺爺的收銀機還老。」
秋穗：「前一個讀的人，大概是在車上讀完的。……然後就這樣留在裡面，幾十年。」
（她的聲音很平。可是指尖在書頁的角上，輕輕停著。）
主角：「你要買嗎？那本。」
秋穗：「……隨便。反正只是一本舊書。」
（嘴上這麼說，她卻把書往懷裡挪了半寸，像是怕被誰先拿走。）
主角：「是是，只是一本舊書。」
（我沒戳破。爺爺耳背，什麼也沒聽見，只在她遞過去的時候，往扉頁上又蓋了一枚新的梅花章——新的和舊的，隔了幾十年，並排在一起。）
（沒有人再說話。夕陽一點一點暗下去，河的聲音漫進店裡來。）
```

### 樣本 3 〔A3 飲料・B2 他遞物・C3 目光沒回到書上・T8 散場〕

```
【盛夏黃昏・河鹿堂】

（自動販賣機買的汽水在手裡出著汗。推開木門，涼下來的空氣裡是舊紙的味道。）
（靠窗那張壞了彈簧的皮沙發上，柏木秋穗抱著一本書，抬起眼睛。）

秋穗：「……佐伯。書店裡不能喝東西。」
主角：「知道啦，還沒開。」
（我在沙發另一頭的木箱上坐下，從袋子裡摸出另一罐，放在她手邊的窗台上。）
主角：「喏。」
秋穗：「……我不喝汽水。太甜。」
主角：「所以那罐是無糖紅茶。」
（她看著窗台上的罐子，安靜了兩秒。）
秋穗：「…………為什麼知道。」
主角：「上次陽溜亭，你把冰淇淋汽水上面的冰淇淋挑掉了。看的人很心痛。」
秋穗：「……那是野上同學擅自加的。」
（她把書合上，抬頭對視了一秒，又低下頭，把左側的頭髮塞到耳後。）
秋穗：「……謝謝。出去再喝。」
主角：「嗯。喝完各自回家。」
秋穗：「……嗯。你先走吧，我還要再看一下下。」
（這麼說著，她的目光卻一直沒有落回書頁上。）
```

### 樣本 4 〔A4 貓・B3 無遞物・C4 要來就來・T10 鉤子〕

```
【盛夏黃昏・河鹿堂】

（結完帳出來，天邊還亮著一線。店門口的石板路上蹲著一隻花貓，正和站在幾步外的柏木秋穗對視。）

主角：「……你們在幹嘛？」
秋穗：「不要出聲。」
（花貓看看她，看看我，尾巴掃了一下，鑽進書店旁的窄巷裡去了。）
秋穗：「……跑了。」
主角：「想摸就去摸啊。」
秋穗：「不摸。」
（她說得很乾脆。）
秋穗：「保持距離就好。……牠有牠的日常，我有我的。互相看一眼，就夠了。」
主角：「哦——所以我也是保持距離就好嗎。」
秋穗：「……你和貓不一樣。」
（說完她自己好像也愣了一下，別開視線。）
秋穗：「……總之，先這樣吧。我回去了。」
主角：「嗯，路上小心。」
（她走出兩步，又停下來，沒有回頭。）
秋穗：「明天……是進新書日。你要來的話，就來。」
（隨口說完，她就走了。蟬在河對岸叫。我低頭看了看手裡蓋著梅花章的書。）
（——要來的話，就來。那大概，就是要我來的意思吧。）
```

主角：「挖寶挖成這樣，籃子要被你掏空了。」
秋穗：「……才兩本。」

主角：「這本呢？看起來很推理。」
秋穗：「……那本的詭計，靠巧合。」
主角：「講得跟讀過一樣。」
秋穗：「讀過。」

秋穗：「……要讀的話，這本。邏輯是乾淨的。」
主角：「哦？那就它了。」
秋穗：「沒什麼特別的。只是剛好在這裡而已。」

## 四、觀察（這次實驗量到了什麼）

1. **殘餘空間是可枚舉的，不是無限的。** 十一個槽釘死後，能變的只剩 A/B/C/D 四條軸、共約 144 個座標。而且四條軸各自都長得像表：話題入口＝地點鉤子×dossier 的交集、遞物方向受階與類型約束、反差格直接取自 05 §三、收尾是 T8/T9/T10 三選一。**下一級固化的對象就是這四張軸表本身。**
2. **同座標重採樣（樣本 1 vs 樣本 5）顯示：座標定死後，變的只剩「字面」。** 兩個樣本的 beat 順序幾乎重合（進店→發現→書話題→她推薦→蓋章→鉤子），差異縮到句子層——這就是真正留給 LLM 的**最小創意區**：那一兩句有靈魂的台詞（「讀完再來」vs「跟我說結局猜對了沒」）。骨架部分完全可查表算掉。
3. **話題入口軸（A）是四軸裡最值得先固化的**：它的候選不是自由發想，而是機械交集——秋穗 dossier 給「舊車票／討厭甜膩／遠觀貓／推理書」，店誌給「梅花章／二手籃／貓／進新書日」，交集即候選池。**換角色或換地點，池子跟著算出來**，不用重想。
4. **反向可當檢核器**：拿任何一段生成稿反推它的 (A,B,C,D) 座標——若反推不出唯一座標（話題漂移、反差格混用、收尾不落在 T8/T9/T10），就是場景寫糊了，打回。
5. **座標之間有相容性梯度**（同 04 §五的組合約束會再長一層）：A3 飲料入口天然接 B2 他遞物；A2 舊車票天然接 B3 無遞物＋T9 定格（安靜的話題配安靜的收尾）。亂配（A2 舊車票＋T10 元氣鉤子）就會怪——這條約束表是本實驗看到、還沒寫下的下一張表。
