# 旁白預算與洩底 —— 旁白層的節流閥

← [README](README.md)｜上層通用表：[05 情緒語域對照](../05_情緒語域對照.md)

> 旁白是含蓄系的**第二聲道**（台詞說表面、旁白洩真心），但它有兩種失敗模式：**太少**（乾、讀不到情緒）和**太多**（說明腔、油）。
> 本檔把「幾拍、放哪、洩什麼」固化成預算表——數字從 [00](00_定點採樣.md) 五樣本歸納，**待大語料驗證**。

## 一、感官細節預算（本定點）

| 位置 | 拍數 | 候選池（店誌 §四＋盛夏） | 規則 |
|------|------|--------------------------|------|
| B0 定場 | **2** | 門鈴／卡卡木門／舊紙味／店內涼 vs 門外蟬鳴／百葉窗條狀光／二樓相撲轉播 | 一聲＋一味（或一光）；**門鈴幾乎必選**——它是「進入河鹿堂時空」的開關 |
| 中段（B3–B5） | 0–1 | 算盤聲／翻頁聲／夕陽移動（影子變長） | 只在**沉默拍**放（她不說話時鏡頭才有空看別處） |
| B7 收尾 | **1** | 河聲漫進來／蟬在對岸叫／光一點一點暗 | **鏡頭必須拉遠**：收尾拍選「比店更大的東西」（河、對岸、天色）——把這一場放回小鎮裡 |

**通則**：同一感官不重複（門鈴響過就不再響）；相撲轉播＝「有人味的安靜」專用、每場 ≤1；**旁白永遠不解說情緒**（「她很開心」＝禁句，開心要用動作洩）。

## 二、洩底動作池（秋穗・本定點）

C 軸反差格的鐵配件：**每個反差瞬間，恰好一個洩底動作**——零個＝反差讀不到（變成她真的冷漠），兩個以上＝說明過度（油）。

| 洩底動作 | 洩的是什麼 | 已配格 |
|----------|-----------|--------|
| 把左側頭髮塞耳後 | 「我要認真看你了」的前奏 | 通用（dossier 小動作） |
| 指尖摩挲書頁角 | focus 在掩飾動搖 | C2 |
| 書往懷裡挪半寸 | 嘴上隨便、其實在乎 | C1 |
| 目光沒落回書頁 | 人沒走心先送客 | C3 |
| 說完自己愣住、別開視線 | 話說得比心快 | C4 |
| （稀有）對視一秒才移開 | 防備放下的一格 | 問句後專用，每場 ≤1 |

**質的規則**：洩底動作必須**小於**它洩的情緒（動作幅度＜情緒幅度）——這就是含蓄的操作型定義。反例：跺腳、掩面、眼眶泛紅（全部＝動作 ≥ 情緒 ＝ 油，[03 護欄](../03_禁語與OOC護欄.md) 事項）。

## 三、旁白視角約束（一人稱的邊界）

旁白＝主角視角，所以旁白**只能寫他觀察得到的**：

```
✓ 「翻頁的手停了一下。」（可觀察）
✕ 「她心裡泛起一絲暖意。」（越權——直接進入她的內心）
△ 「像是怕被誰先拿走。」（「像是」＝他的解讀，合法；但每場 ≤2，解讀多了＝自作多情）
```

這條約束**免費送一個檢核器**：掃旁白裡有沒有「她覺得／她心想／她其實」——命中即打回，純字串規則、零成本。

## 本檔引出的問題（詳見 [90](90_問題與討論.md)）

- 預算數字（2 拍／0–1 拍／1 拍）樣本只有五個——去 34 篇劇本＋52 短場景統計真實分布（丙-2）
- 「動作幅度＜情緒幅度」能否量化？（給動作和情緒各標強度分級，檢核變可計算）（丙-3）
- 心聲（——起手）和旁白（（）括號）的分工邊界要不要成文（乙-2）
