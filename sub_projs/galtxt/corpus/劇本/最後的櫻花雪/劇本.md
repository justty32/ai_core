# 最後的櫻花雪

> **類型**：短篇劇本章節（含選擇肢分支，兩線收束同一結尾）
> **場景**：舊校舍天台 →〔分支〕河鹿堂／陽溜亭 → 霞川河堤 → 木造車站
> **角色**：主角（男）、秋穗（女，新生代表）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
【縣立日和高等學校・舊校舍天台】
（四月的風夾雜著櫻花瓣的餘香，掠過空曠的鐵絲網。）
（牆角堆放著廢棄的課桌椅，被陽光曬得微微發熱，散發出陳舊的木頭氣味。）
（鏽蝕的圍欄外，可以俯瞰整座日和町——靜靜流淌的霞川，以及河畔兩側漸漸由粉轉綠的櫻並木。）
主角：「呼……果然這裡最安靜了。」
（推開那扇從不上鎖的鐵門，本以為只有自己，卻看見一個熟悉的背影立在護欄前。）
（那是同班的秋穗。她正微微踮起腳尖，任憑風吹亂她黑色的短髮。）
秋穗：「啊，是你。你也來這裡躲避開學典禮的吵鬧嗎？」
主角：「嗯，禮堂裡實在太悶了。話說回來，你不用去代表新生致詞？」
秋穗：「早就結束啦。況且，比起聽校長冗長的訓話，我更想看今年的最後一場櫻花雪。」
主角：「畢竟是新生代表，校長肯定對你抱有很大期待吧。」
秋穗：「所以才想逃跑啊。比起那些期待，我更喜歡這種沒人打擾的日常。」
（她轉過身，手裡拿著一本泛黃的舊書，封面上印著老書店『河鹿堂』的梅花章。）
秋穗：「吶，你看。從這裡望下去，霞川像一條綠色的帶子呢。」
主角：「確實。再過幾天，櫻花落盡，夏天就要來了吧。」
秋穗：「是啊……時間過得真快。對了，待會放學，你打算直接回家嗎？」

　　▷ 選項A：邀她一起去老書店「河鹿堂」
　　▷ 選項B：邀她一起去定食屋「陽溜亭」

【選項A的後續】
主角：「如果沒事的話，要不要去『河鹿堂』看看？我看你手上的書，好像是那裡的。」
秋穗：「被你發現了？這本小說我剛好看到最後一章，正想去還呢。」
（她露出淡淡的微笑，將書抱在胸前，眼神亮了起來。）
秋穗：「那走吧，聽說老闆最近進了一批有趣的二手推理小說。」
（我們沿著杜坂並肩走下山，午後的陽光穿過樹蔭，在她的肩頭灑下斑駁光影。）

【選項B的後續】
主角：「肚子有點餓了，要不要去『陽溜亭』吃個下午茶？今天的特餐好像是紅豆湯圓。」
秋穗：「欸？聽起來真誘人。正好我也覺得肚子空空的。」
（她微微偏過頭，臉頰似乎因為風吹而有些泛紅。）
秋穗：「那可不能落後了，要是被其他人搶先點完，我可是會生氣的。」
（我們沿著杜坂並肩走下山，空氣裡似乎也多了些甜甜的、讓人放鬆的溫度。）

【霞川河堤】
（斜陽將我們的影子拉得很長，霞川的河水在夕陽下泛著細碎的金光。）
（微風拂過，兩側櫻堤落下了最後的櫻花雨，輕輕飄落在水面上，打著旋兒流向遠方。）
（鎮上的商店街傳來遠遠的喧囂，但河堤上只有我們踩在碎石路上的腳步聲。）
秋穗：「結果，還是來到了河堤呢。」
主角：「因為回車站剛好順路。不過，今天的風真的很舒服。」
（秋穗停下腳步，看著飛舞的落櫻，眼神裡帶著一絲不易察覺的落寞。）
秋穗：「新學期開始了，大家都在往前走。有時候，會覺得有點害怕呢。」
主角：「害怕什麼？」
秋穗：「害怕自己停留在原地，也害怕那些熟悉的風景，不知不覺就變了樣。」
主角：「沒關係的。」
主角：「就算風景變了，我們不是還能像現在這樣，一起走在這條路上嗎？」
（秋穗微微一怔，隨後轉過頭看著我，眼角的落寞漸漸消散，化作溫緩的笑意。）
秋穗：「嗯……你說得對。謝謝你。」
主角：「走吧，電車快來了。」
（她輕輕點頭，與我並肩踏上歸途，兩人的影子在水泥路上重疊在一起。）
（最後的櫻花瓣落在她的髮梢，我們迎著晚風，一步步走向那座老舊的木造車站。）
```
