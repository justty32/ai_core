## 一、場景條件（固定 canon——二十流一律不變）

**人事時地物：**

| 軸 | 值 |
|----|----|
| **時** | 傍晚・斜陽（由午後推進）；電車將到＝離場催促 |
| **地** | 霞川河堤・碎石路；遠處商店街喧囂 ↔ 河堤靜（＝兩人獨處） |
| **人** | 主角（男）＋秋穗（女・新生代表）・並肩・剛熟的同班 |
| **物・母題** | **最後的櫻花雨**（花期正式落盡＝季節時鐘走到終點）、泛金河水、兩人影子、（將到的）電車、老木造車站 |

**情緒弧（秋穗，唯一真正演化的狀態線）：** 自在 →〔轉折〕落寞（不易察覺，需主角留意才接得住）→〔核心〕吐露怕停滯／怕熟悉風景變樣 →〔釋懷〕落寞消散、溫緩笑意（★好感度 +1 躍升）。

**這場在整個故事中的作用：**

1. **全劇情感高潮**——前面（天台相遇、分支閒逛）都是鋪陳，情感在此第一次真正externalize。
2. **A/B 兩線的收束護欄**——不管選河鹿堂還是陽溜亭，都因「回車站順路」繞到河堤，人事時地物在此**重新合一**；台詞層要能承接兩種來路（帶書／未帶書）。
3. **主旨落點**——把全劇的「變動母題」（櫻花將盡→粉轉綠→最後櫻花雨）翻轉成正面回答：**風景會變，一起走不變**。
4. **季節母題閉合**——「最後的櫻花雪」在此正式落盡，標題於此兌現。
5. **關係推進節點**——收場時兩人比開場更近（並肩→影子重疊），為後續好感線埋錨。

**不可違反的骨架（每一流都要走完）：**
① 抵河堤（傍晚・落櫻）→ ② 秋穗情緒轉折（落寞，明說或動作洩底皆可）→ ③ 秋穗吐露某種「怕變／怕停滯」的不安 → ④ 主角**接住**並給出「陪伴／連結」的托底 → ⑤ 秋穗釋懷 → ⑥ 收於「走向車站」。

## 二、變體軸（二十流之間怎麼不同）

固定事實層之上，六條可調的軸——二十流各是這些軸的一種組合（非全笛卡爾，取有代表性的 20 點覆蓋空間）：

| 軸 | 取值 |
|----|------|
| **A 誰起頭情緒轉折** | 秋穗主動吐露／主角先察覺追問／由景（落櫻）自然引出／沉默中先開口 |
| **B 秋穗的表達度（反差深淺）** | 直說怕／玩笑包裝／純動作洩底（幾乎不說）／說反話嘴硬 |
| **C 主角的回應風格** | 直球安慰／先玩笑再轉正／不說教只承諾陪伴／反問引她自己想通／具體約定 |
| **D 母題載體** | 櫻／河水／影子／電車時刻／風／季節 |
| **E 節奏篇幅** | 話密／留白多／極簡／綿長 |
| **F 收尾情緒色** | 甜暖／靜淡／微酸（惆悵未全消）／明快（拉回日常） |

> 讀法：每流標「座標」＝〔A｜B｜C｜D｜E｜F〕＋一句意圖，再給台詞流。

## 三、二十種台詞流

### 01　基準（原案骨幹）
〔秋穗主動｜直說怕｜直球安慰｜櫻｜適中｜靜淡〕　*把 R04–R14 收成最平實的一版，當對照原點。*
```
（斜陽把兩人的影子拉長，河水泛著細碎的金光。兩側櫻堤落下最後的櫻花雨。）
秋穗：「結果，還是來到了河堤呢。」
主角：「回車站剛好順路。……今天的風真舒服。」
（她停下腳步，看著飛舞的落櫻，眼神裡有一絲不易察覺的落寞。）
秋穗：「新學期開始了，大家都在往前走。有時候……會有點害怕呢。」
主角：「害怕什麼？」
秋穗：「怕自己停在原地。也怕那些熟悉的風景，不知不覺就變了樣。」
主角：「沒關係的。就算風景變了，我們不是還能像現在這樣，一起走在這條路上嗎？」
（她微微一怔，轉過頭，落寞化作溫緩的笑意。）
秋穗：「嗯……你說得對。謝謝你。」
主角：「走吧，電車快來了。」
```

### 02　玩笑包裝
〔秋穗主動｜玩笑遮｜先玩笑再轉正｜櫻｜適中｜明快〕　*她用玩笑把不安藏起來，主角先陪笑、再一句戳到底。*
```
（河堤上只剩兩人的腳步聲，最後的櫻花雨落在泛金的水面上。）
秋穗：「你說，櫻花每年都謝，它會不會覺得自己很沒用？」
主角：「問櫻花的心理，你也想太多了吧。」
秋穗：「……大概是被新學期傳染的。大家都在往前衝，就我還站在原地看花。」
主角：「站在原地看花的人，才看得到最後一場櫻花雪啊。」
（她愣了一下，笑出聲，肩上的緊繃鬆了。）
秋穗：「歪理。……但不討厭。」
主角：「走吧，電車要來了，再看下去你要在這裡紮根了。」
（她輕輕點頭，並肩踏上歸途。）
```

### 03　純動作洩底（極簡・留白）
〔沉默先開口｜幾乎不說｜只一句托底｜櫻｜極簡留白｜靜淡〕　*她的怕全在停步與凝視裡，主角讀懂沉默。*
```
（傍晚的河堤，最後的櫻花瓣一片片落進金色的河水。）
（秋穗忽然停下，很久沒有說話，只看著落櫻打旋流遠。）
主角：「……在想很遠的事？」
秋穗：「嗯。……有點怕。」
主角：「我在。」
（就這樣。她看了他一眼，落寞散了。）
秋穗：「……走吧。」
（兩人並肩，影子在碎石路上疊在一起，走向車站。）
```

### 04　嘴硬反話
〔秋穗主動｜說反話｜不拆穿只陪走｜櫻｜適中｜微酸〕　*她嘴上說不怕，主角不揭穿，順著她給台階。*
```
（落櫻紛飛，河水在斜陽下泛金，商店街的喧囂遠遠傳來。）
秋穗：「新學期什麼的，我才不會不安呢。」
主角：「嗯，我知道你不會。」
秋穗：「……只是覺得，這些風景說變就變，有點討厭而已。」
主角：「那，就算它變了——」
（他頓了頓。）
主角：「不怕的你，也可以繼續走這條路。我順路。」
（她抿著嘴，別過臉，可眼角的落寞悄悄退了。）
秋穗：「……哼，隨便你。走了，電車快到。」
```

### 05　反問引導
〔主角先察覺｜半吐｜反問引她自己想通｜這條路｜適中｜靜淡〕　*主角不給答案，用一個問題讓她自己翻轉。*
```
（河堤靜得只剩腳步聲，最後的櫻花雨落在水面。）
（主角先注意到她放慢了腳步，眼神黯了下來。）
主角：「怎麼了，突然不說話。」
秋穗：「……在想，熟悉的東西會不會有一天全變了樣。有點怕。」
主角：「那我問你——這條河堤，明年還在嗎？」
秋穗：「……在吧。」
主角：「我們還能一起走嗎？」
（她怔了怔，望向他，落寞慢慢化開。）
秋穗：「……你這個人，真狡猾。嗯，謝謝。」
主角：「走吧，電車來了。」
```

### 06　具體約定（把抽象的怕錨在重逢）
〔秋穗主動｜直說怕｜具體約定｜季節｜適中｜甜暖〕　*主角不講道理，給一個「明年再來看」的約定。*
```
（斜陽・落櫻・泛金的霞川，兩人踩在碎石路上。）
秋穗：「今年的櫻花，就這樣謝完了呢。……有點捨不得。」
主角：「明年還會開。」
秋穗：「可是明年的我們，說不定就不一樣了。大家都在往前走……」
主角：「那就說好——明年這時候，還來這條河堤看最後一場櫻花雪。」
（她抬起頭，眼裡的落寞被什麼點亮了。）
秋穗：「……一言為定？」
主角：「一言為定。走吧，電車快來了。」
（她點頭，笑意溫緩，並肩走向車站。）
```

### 07　母題＝河水（變的是水、不變的是河）
〔由景引出｜半吐｜以河水比喻托底｜河水｜適中｜靜淡〕
```
（河水在夕陽下泛金，載著最後的落櫻打旋流向遠方。）
秋穗：「你看那些花瓣，一下就被水沖走了。……我有時候會怕，自己也這樣被沖走。」
主角：「花瓣是被沖走了。可河還在這裡，明天也在。」
秋穗：「……河？」
主角：「變的是水，不變的是河床。我們走的這條路，就是河床那種。」
（她望著他，愣了一下，落寞漸漸退去。）
秋穗：「嗯……這樣想，好像就不那麼怕了。謝謝你。」
主角：「走吧，電車來了。」
```

### 08　母題＝影子（並肩重疊）
〔主角先察覺｜純動作洩底｜不說教只陪伴｜影子｜留白｜甜暖〕
```
（斜陽把兩人的影子在碎石路上拉得很長。最後的櫻花雨無聲地落。）
（秋穗停下，盯著自己被拉長的影子出神。）
主角：「影子被拉這麼長，像要走到河對岸去了。」
秋穗：「……人也是。大家都往前走，我怕自己被落下。」
（主角沒說話，只往她身邊挪了半步，兩人的影子疊在了一起。）
主角：「這樣就不會落下了。並排的。」
（她看著地上重疊的影子，唇角慢慢彎起。）
秋穗：「……嗯。走吧，電車快到了。」
```

### 09　母題＝電車時刻（恆常）
〔秋穗主動｜直說怕｜以電車恆常托底｜電車｜適中｜明快〕
```
（遠處傳來商店街的喧囂，河堤上落櫻紛紛，斜陽將盡。）
秋穗：「新學期一開始，就覺得時間推著人跑。……有點喘，有點怕。」
主角：「你聽，電車的聲音。」
秋穗：「……嗯？」
主角：「它每天這個點都來，風雨無阻。我們也可以——每天這個點，走這條河堤。」
秋穗：「把自己過成電車時刻表，虧你想得出來。」
（她笑了，眼裡的落寞散了。）
主角：「至少準時可靠。走吧，正主快到站了。」
```

### 10　母題＝風（吹走櫻、也把人吹在一起）
〔由景引出｜半吐｜先玩笑再轉正｜風｜適中｜靜淡〕
```
（晚風拂過，捲起兩側櫻堤最後的花雨，打著旋兒落向水面。）
秋穗：「風把花全吹走了。……也不知道會把我們吹到哪裡去。」
主角：「吹到車站啊，很近。」
秋穗：「我說的是……以後。大家散開、風景變樣的那種以後。」
主角：「同一陣風吹的，就算散開，也是往同個方向。至少現在，吹著我們一起走。」
（她怔了怔，望向他，落寞被晚風帶走了些。）
秋穗：「……你偶爾也會說正經話嘛。謝謝。走吧。」
```

### 11　極簡（四拍到底）
〔沉默先開口｜純動作洩底｜一句托底｜櫻｜極簡｜靜淡〕　*最短的一版，逼出骨架下限。*
```
（傍晚河堤，最後的櫻花雨。秋穗停步，看花，沉默。）
秋穗：「……怕它們變。」
主角：「有我在，一起走。」
（她轉頭，笑了。並肩走向車站。）
```

### 12　綿長（話密・鋪陳壓力）
〔秋穗主動｜直說怕｜傾聽後長答｜季節｜綿長話密｜微酸〕　*她說得多，把新生代表的壓力鋪開；主角聽完再長答。*
```
（斜陽・泛金河水・最後的落櫻，河堤只剩兩人的腳步聲。）
秋穗：「當上新生代表之後，大家看我的眼神就變了。老師期待、同學觀望，好像我得一直往前、不能停。」
秋穗：「可我明明只想像現在這樣，安安靜靜看一場櫻花雪。……新學期才剛開始，我就已經在怕了。」
秋穗：「怕自己停在原地被落下，也怕有一天回頭，這些熟悉的風景全變了樣，連這條河堤都認不得。」
（主角沒有馬上答，等她說完，看著同一片落櫻。）
主角：「風景大概真的會變。花會謝，河會漲，代表會換人當。」
主角：「可是你回頭的時候，只要還有個人陪你一起認這些風景——變了也認得回來。這種事，我可以。」
（她怔怔看著他，積在眼角的落寞，一點一點化成溫緩的笑。）
秋穗：「……說得真好。謝謝你，真的。走吧，電車快來了。」
```

### 13　微酸收尾（惆悵未全消）
〔秋穗主動｜半吐｜承諾陪伴｜櫻｜適中｜微酸〕　*釋懷但不徹底，主旨落在「一起怕也沒關係」。*
```
（最後的櫻花雨落在泛金的水面，斜陽正一點點沉下去。）
秋穗：「……花謝了就是謝了，明年的不是今年這一場。有些東西，是真的回不來的吧。」
主角：「嗯。回不來的，就是回不來。」
秋穗：「你倒是誠實。」
主角：「但下一場櫻花雪，我還想跟你一起看。就算怕它又謝，一起怕，好像也沒那麼難熬。」
（她沒有笑得很開，只是輕輕地、有點濕潤地彎了下唇。）
秋穗：「……嗯。一起怕。……走吧，電車來了。」
（兩人的影子在碎石路上疊著，走向車站。）
```

### 14　主角先察覺（順序倒置）
〔主角先察覺｜純動作洩底｜直球安慰｜櫻｜適中｜靜淡〕　*不等她開口，主角先接住「不易察覺」的那一絲。*
```
（河堤靜，落櫻紛飛。走著走著，主角發現身邊的腳步慢了下來。）
主角：「……你停下來了。」
秋穗：「有嗎。」
主角：「眼神也飄到落櫻上去了。在怕什麼？」
（她愣了愣，被說中，輕輕吐氣。）
秋穗：「……怕熟悉的東西變樣。新學期一到，就特別明顯。」
主角：「那我盯著點。哪天風景要變，我提前跟你說一聲，我們一起走過去。」
（落寞化開，她轉頭看他，笑意溫緩。）
秋穗：「嗯……謝謝。走吧，電車快到了。」
```

### 15　落櫻自然引出（物先於言）
〔由景引出｜半吐｜反問引導｜櫻｜適中｜甜暖〕　*一片花瓣落在她手心，情緒由物觸發。*
```
（晚風送來最後的櫻花雨。一片花瓣，恰好落在秋穗攤開的掌心。）
秋穗：「……最後一瓣了吧。接住了，也留不住。」
主角：「留不住的，就是這種花瓣好看的地方。」
秋穗：「你不怕嗎？怕這些好看的、熟悉的，說沒就沒。」
主角：「你手裡不是接住了嗎？只要有人一起接，沒了一瓣，還有下一瓣。」
（她低頭看著掌心的花瓣，又看看他，眼裡的落寞化成暖意。）
秋穗：「……嗯。那下一瓣，也一起接。走吧，電車來了。」
```

### 16　秋穗主導收束（主角陪襯）
〔秋穗主動｜直說怕→自己想通｜主角只托底｜這條路｜適中｜靜淡〕　*換她自己走完落寞→釋懷，主角只給一個支點。*
```
（傍晚・河堤・最後的落櫻，兩人並肩。）
秋穗：「結果還是來到了河堤。……剛剛一路上，我一直在怕。怕新學期、怕大家往前走、怕風景變。」
主角：「嗯。」
秋穗：「可是走到這裡我突然想通了——會變的東西，我攔不住。」
主角：「那攔不住的，怎麼辦？」
秋穗：「……找個會一起走的人，就不怕了。」
（她說完，轉頭看他，自己先笑了。）
主角：「這不是想明白了。走吧，電車快到。」
```

## 四、給固化的觀察（這 20 流逼出什麼）

- **canon 與風格的界線很清楚**：六拍骨架（抵達→轉折→吐露→接住→釋懷→走向車站）二十流一律守；差異全在**表達度（B）＋回應風格（C）＋母題載體（D）**三軸——證實「事實／敘事風格分離」在此場景成立，realizer 的自由度就是這三軸。
- **母題載體可替換而主旨不變**：櫻／河水／影子／電車／風／季節，任一都能承載「變 vs 不變」的主旨（見 07/08/09/10）——母題是**可插拔的隱喻槽**，不是硬事實。
- **洩底可以零台詞**：03/11/20 幾乎不靠秋穗說「怕」，用停步／凝視／慢半拍就完成情緒轉折——印證知識視圖模型裡「反差＝表面與 hides 不一致」可由**動作**而非言語達成。
- **收束護欄穩固**：二十流都能自然收在「走向車站」，且能同時承接 A 線（帶書，見 19）與 B 線（未提書）——證明河堤作為兩線收束點的設計是台詞層可承接的。
- **好感 +1 節點可調亮度**：從 20（幾乎不可見）到 18（近告白邊緣），同一躍升可用不同亮度實現——好感度事件是**連續量**，不是開關。
- **下一步**：若要進 gen 的句庫，這 20 流可標成 realizer 候選（act 相同、字面不同），配合 B/C/D 三軸當 realizer 的座標——正好接上 gen_v1「realizer 敘事風格層」那條待議線。
