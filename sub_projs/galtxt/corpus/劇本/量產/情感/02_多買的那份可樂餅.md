# 多買的那份可樂餅

- 類型：青春日常／情感轉折／鬧彆扭與和好
- 場景：定食屋「陽溜亭」門口、霞川 櫻並木河堤
- 角色：岡田 遙（青梅竹馬、同校不同班、嘴快）、佐伯悠（主角）

```
【商店街・定食屋「陽溜亭」門口（放學後・傍晚）】

（陽溜亭的招牌燈剛亮，玻璃門後飄出炸物的香。我來買晚飯，卻在門口撞見一張臭臉。）

（岡田遙。從日和中學校一路同校上來的青梅竹馬，隔壁班。此刻她抱著手臂，背對著我，明擺著在生氣。）

佐伯悠：「……你站這幹嘛，堵門？」

岡田遙：「沒堵。路是大家的。」

（連正眼都不給。這語氣我太熟了——今天早上，我確實忘了一件事。）

（昨天她說今天是她生日，要我陪她走一趟舊鐵道遺址。我隨口應了，然後被美術社的作業纏住，整個放學都泡在畫室，把這事忘得一乾二淨。）

佐伯悠：「那個……早上的事。」

岡田遙：「什麼事？我不記得有什麼事。」

（她記得清清楚楚。她一生氣就這樣，把話反著說，等你自己去踩地雷。）

（陽溜亭的老闆娘從門後探頭。）

老闆娘：「小遙，你要的兩份可樂餅好囉——咦，悠也來啦。」

佐伯悠：「兩份？」

（岡田遙的肩膀僵了一下。她本來，是打算買兩份，去舊鐵道那邊等我的。）

【霞川・櫻並木河堤（暮色・並肩走）】

（她提著那袋可樂餅，二話不說往河堤走，我只好跟上。櫻並木的葉子在暮色裡沙沙響，她走得飛快，像要把我甩掉，卻又沒真的甩。）

岡田遙：「你不用跟。」

佐伯悠：「順路。」

岡田遙：「哪裡順路，你家在反方向。」

（被抓包了。我閉嘴，繼續跟。走到河堤中段那張舊長椅，她一屁股坐下，把袋子往中間一擱。）

岡田遙：「……涼了。」

佐伯悠：「可樂餅涼了也好吃。」

岡田遙：「我說的不是可樂餅。」

（她盯著河面。霞川的水把最後一點天光揉碎了。我知道她指的是什麼。）

──────────────────
【選擇肢】她把彆扭攤開了一角，你怎麼哄？
　A：正經道歉，把忘掉的理由老實說
　B：不硬碰，先陪她把這口氣消掉
──────────────────

《A：老實道歉》

佐伯悠：「對不起。我被作業纏住，把今天的事忘了。這是我的錯，不推。」

（她沒想到我道得這麼直，愣了一下。）

岡田遙：「……你至少該說『忙到忘了』，聽起來比較不像故意的。」

佐伯悠：「不。就是忘了。忙不是理由。」

（她別過臉，肩膀卻鬆了下來。過了半晌，聲音悶悶的。）

岡田遙：「……知道錯就好。誰要你想那麼久的藉口了。」

（她拆開一份可樂餅，掰成兩半，把大的那半塞到我手裡。這就是她的原諒了。）

《B：先陪她消氣》

（我沒急著解釋。只是坐下來，陪她一起看河。)

佐伯悠：「今天的霞川，比早上安靜。」

岡田遙：「……你早上根本沒看霞川，你在畫室。」

佐伯悠：「所以現在補看。陪你一起。」

（她噎了一下，本來鼓著的腮幫子，慢慢洩了氣。）

岡田遙：「……你這個人，真的很狡猾。生氣都生不下去。」

佐伯悠：「生日快樂，遙。遲到的。」

（她猛地轉過頭，眼眶有點紅，卻硬是瞪著我不讓它掉。)

岡田遙：「……哼。記著，你欠我一整個下午。」

（她拆開可樂餅，掰了大的那半給我。）

【舊鐵道遺址・生鏽的月台（入夜）】

（消了氣，她自己說想去原本要去的地方。舊鐵道的月台早沒了列車，只剩一條長草的軌，鏽成暗紅色。我們並肩坐在月台邊緣。）

岡田遙：「小時候你也是這樣。答應的事老是忘。」

佐伯悠：「後來不都補回來了嗎。」

岡田遙：「……勉強及格。」

（她咬著可樂餅，含糊地補了一句，聲音輕得幾乎要被夜風吹走。）

岡田遙：「不過——今年這個生日，等到的人有來，就算數了。」

佐伯悠：「……明年別讓我猜哪天。直接寫在我手上。」

岡田遙：「才不要。你要自己記得，才有意義。」

（她說得理直氣壯，卻悄悄把最後一口可樂餅，分了我一半。鏽軌的另一頭，町裡的燈一盞一盞亮起來。）
```
