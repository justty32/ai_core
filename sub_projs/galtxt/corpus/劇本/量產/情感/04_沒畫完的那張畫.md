# 沒畫完的那張畫

- 類型：青春日常／情感轉折／心結的坦白・鼓起勇氣的一步
- 場景：日和高校 美術教室、杜坂 半山的老櫻
- 角色：藤井 睦美（同美術選修、沉靜、手很巧）、佐伯悠（主角）

```
【日和高校・美術教室（放學後・斜陽）】

（美術教室在老木造校舍的西側，斜陽從高窗照進來，把顏料的味道曬得暖暖的。選修的人走得差不多了，只剩我和藤井睦美。）

（藤井睦美。同修美術，沉靜，手很巧，畫的水彩總是乾淨。可最近一個月，她的畫架上永遠是同一張——半山那棵老櫻，畫了一半，就再也沒動過。）

佐伯悠：「這張……你放好久了。」

藤井睦美：「……嗯。」

佐伯悠：「卡住了？」

藤井睦美：「不是卡住。是不敢畫完。」

（她的筆停在調色盤上，沒有沾水。這話說得很輕，卻明顯藏著別的東西。）

佐伯悠：「不敢？」

藤井睦美：「……這棵樹，是我姊教我畫的。她以前也在這間教室。」

（她第一次提起姊姊。我沒接話，等她自己往下說。）

藤井睦美：「她畢業那年，說要一起去把這棵樹畫完。後來她去了很遠的學校，這張就一直停在這裡。我怕……畫完了，就真的只剩我一個人畫了。」

（斜陽移了一格，落在那張沒畫完的櫻花上。原來她這一個月的停筆，停的不是畫，是一句沒說出口的話。）

【杜坂・半山的老櫻（隔日・放學後）】

（隔天我硬把她拉出教室，說光看畫架沒用，去看真的樹。杜坂爬到一半，那棵老櫻就站在轉角，葉子綠得發亮。她仰頭看了很久。）

藤井睦美：「……我上次這樣抬頭看它，是跟姊一起。」

佐伯悠：「現在也不是你一個人。」

（她低下頭，手指絞著制服的衣角。森林綠的裙擺被山風掀了掀。）

藤井睦美：「佐伯，我一直想問你一件事，可我沒膽。」

佐伯悠：「說。」

藤井睦美：「那張畫……如果我畫完，你願不願意，跟我一起署名。這樣它就不是『只剩我一個人』的畫了。」

（她說完，臉埋得更低。這對她來說，大概是鼓足了全部勇氣的一步。）

──────────────────
【選擇肢】她把心結和請求一起交了出來，你怎麼回應？
　A：正面接下這一步，答應與她並肩署名
　B：先替她把「怕孤單」這個結，輕輕解開
──────────────────

《A：接下這一步》

佐伯悠：「好。署名可以。不過有個條件。」

藤井睦美：「……條件？」

佐伯悠：「畫完這張以後，下一張換我起頭，你陪我畫完。輪流。這樣就沒有『最後一張』了。」

（她抬起頭，眼睛亮了一下，像被說中了心底最怕的那件事。）

藤井睦美：「……你是說，一直畫下去。」

佐伯悠：「嗯。畫到畢業，再畫到以後。」

藤井睦美：「……那，說定了。」

（她伸出小指。我勾上去。老櫻的葉子在頭頂沙沙地響，像替我們蓋了個章。）

《B：先解開她的結》

佐伯悠：「署名之前，先問你一句。」

藤井睦美：「嗯？」

佐伯悠：「你姊去了遠的學校，可她有跟你斷了嗎？」

藤井睦美：「……沒有。她上週還寄了明信片。」

佐伯悠：「那這棵樹就沒有『只剩你一個』過。它連著她，也連著現在站在這裡的我。」

（她怔怔地看著我，眼眶慢慢紅了，卻是鬆開的那種紅。）

藤井睦美：「……我好像，把自己困在一個不存在的角落了。」

佐伯悠：「所以走出來。畫完它。要署名，我陪你署。」

藤井睦美：「……嗯。謝謝你，把我拉出來。」

（她抹了抹眼角，第一次在我面前，笑得沒有防備。）

【美術教室・畫架前（暮色・完成）】

（回到教室，天已經暗了一階。她重新在那張畫前坐下，這次筆尖沾了水，很穩。我在旁邊替她遞顏料。）

藤井睦美：「最後這幾筆，我想自己畫。你在旁邊看著就好。」

佐伯悠：「看著。」

（她一筆一筆把老櫻的花補上，動作輕，卻不再猶豫。落下最後一筆時，她長長吐了口氣，像卸下了扛很久的東西。）

藤井睦美：「畫完了。……原來畫完，也沒有想像中那麼可怕。」

佐伯悠：「因為不是一個人畫完的。」

（她在畫的角落，慢慢寫下自己的名字，然後把筆遞給我，指了指旁邊的空位。）

藤井睦美：「輪到你了。」

（我接過筆，在她的名字旁邊，寫下我的。暮色裡的美術教室很靜，只有筆尖劃過紙面的沙沙聲，和窗外杜坂上那棵、終於被畫完的老櫻。）
```
