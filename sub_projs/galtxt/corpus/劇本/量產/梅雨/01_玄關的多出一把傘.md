# 玄關的多出一把傘

- 類型：青春日常／梅雨／共傘
- 場景：日和高校 昇降口玄關、杜坂石階下
- 角色：宮下 澪（同班、圖書委員、話少）、佐伯悠（主角）

```
【日和高校・昇降口玄關（放學後・大雨）】

（梅雨的雨下得毫無誠意，卻又收不了手。我站在玄關的傘架前，把手伸進自己那格——空的。）

（早上分明帶了。八成是被哪個急著回家的傢伙順手抽走了。）

佐伯悠：「……這下好了。」

（雨簾把整座杜坂糊成一片灰白，石階上的水正一級一級往下淌。就這樣衝下去，到家也差不多變成一條落湯狗。）

（玄關只剩我一個。不對——身後傳來換鞋的聲音。）

宮下澪：「你也沒帶傘？」

（宮下澪。同班，圖書委員。平時安靜到我幾乎沒聽過她主動開口。她抱著一把收好的深藍色長傘。）

佐伯悠：「帶了。被人拿走了。」

宮下澪：「……原來如此。」

（她低頭看了看自己的傘，又看了看外面的雨，像在做一道很難的數學題。）

宮下澪：「順路的話。」

佐伯悠：「欸？」

宮下澪：「我家在霞川那邊。你呢。」

佐伯悠：「……同方向。」

（她沒再多說，只是把傘往前遞了半步，意思再明白不過。）

【杜坂・石階下（雨中・共傘）】

（一把傘要遮兩個人，總得有人肩膀濕。她個子比我矮，傘自然而然被我接了過來，往她那側偏。）

宮下澪：「你會淋到。」

佐伯悠：「一邊肩膀而已，死不了。」

（她沒反駁，只把抱在胸前的書包又抱緊了些。雨打在傘面上，劈啪劈啪，反而讓沉默不那麼尷尬。）

（走到石階轉角，一輛自行車濺起一道水花。我下意識把她往內側帶了半步。）

（她愣了一下，沒說話。傘下的空氣好像忽然變窄了。）

【選擇肢】要不要找點話說？
　A：問她剛才在圖書室看什麼書
　B：什麼都不說，安靜走
──────────────────

《A：問她在看什麼書》

佐伯悠：「你剛才……在圖書室看得很入神。什麼書？」

宮下澪：「……你有看到。」

（她耳根微微紅了，大概沒料到有人注意。）

宮下澪：「梅雨季的短篇集。裡面每篇都下著雨。」

佐伯悠：「特地挑下雨的看？」

宮下澪：「下雨的時候看下雨的故事，會覺得……自己不是一個人在淋。」

（我一時語塞。這話從她嘴裡說出來，安靜得像雨本身。）

《B：安靜走》

（我沒開口。她好像也不打算開口。）

（但奇怪的是，這沉默一點也不難熬。傘面的雨聲、她書包上小吊飾偶爾的輕響、我們踩過同一攤水的節奏——像有誰替我們把多餘的話都省下了。）

（走了整整一條坡，她才輕輕說了一句。）

宮下澪：「……這樣也不錯。」

（我不太確定她指的是雨，還是別的。）

【霞川・河堤旁（分岔路口）】

（到了該分開的路口。雨還沒停，天卻亮了一階。）

宮下澪：「傘，明天還我就好。」

佐伯悠：「欸，你不用了？」

宮下澪：「跑幾步就到了。」

（她把傘連著柄一起塞進我手裡，轉身就往雨裡小跑，深藍色的校裙很快被水點染深。）

宮下澪：「明天……記得帶自己的傘。」

（她回頭補了這句，難得地，嘴角是彎的。）

佐伯悠：「……知道了。」

（我撐著那把不屬於我的傘，站在原地看她跑遠。手心還留著傘柄的餘溫。）

（明天要還傘——這理由，好像也沒那麼壞。）

（梅雨還會下很久。而我，忽然不那麼討厭這個季節了。）
```
