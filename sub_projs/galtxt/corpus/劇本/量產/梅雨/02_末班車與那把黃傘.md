# 末班車與那把黃傘

- 類型：青春日常／梅雨／雨中車站
- 場景：日和町車站 月台候車室、站前小巷
- 角色：小早川 千夏（鄰鎮通學、田徑社、開朗）、佐伯悠（主角）

```
【日和町車站・候車室（傍晚・雨）】

（梅雨的傍晚來得早。老木造車站的暖黃燈一盞盞亮起，把雨絲照成一條條金線。）

（我在美術教室多留了一會，錯過了平常那班車。地方支線的末班車很早，這一等，就要等到最後一班。）

（候車室的長椅上，還坐著一個人。）

小早川千夏：「哦——是佐伯同學。你也趕末班？」

（小早川千夏。隔壁班，田徑社的。跑得快、嗓門也亮，走廊上老遠就能認出她的笑聲。此刻她一條腿隨意晃著，運動包擱在膝上。）

佐伯悠：「畫到忘了時間。你呢？練到這麼晚？」

小早川千夏：「雨天不能跑外場，改跑階梯跑到腿軟。」

（她拍拍小腿，齜牙咧嘴地笑。膝蓋上還貼著一塊擦傷的膠布。）

小早川千夏：「你看，杜坂那石階害的。冬天滑、梅雨也滑，簡直跟我有仇。」

佐伯悠：「你倒是天天跟它較勁。」

小早川千夏：「輸給一段坡，很不甘心欸。」

（她說得理所當然，我卻莫名覺得，這種笨拙的認真挺好看。）

【月台（末班車前・雨勢轉大）】

（廣播響起，末班車誤點五分。雨忽然大了，斜著往月台簷下潑。）

（她縮了縮腳，這才發現什麼似地翻起運動包。）

小早川千夏：「……啊。」

佐伯悠：「怎麼？」

小早川千夏：「傘。早上晾在陽台忘了收。」

（她乾笑，把一把只剩骨架的破折傘舉起來——那玩意兒撐開大概比不撐還慘。）

（我看了看自己手裡那把還算完好的黃色雨傘。）

──────────────────
【選擇肢】
　A：把傘借她，自己想辦法
　B：提議一起走，送她到轉車的小巷
──────────────────

《A：把傘借她》

佐伯悠：「拿去。我跑得沒你快，但淋一段還行。」

小早川千夏：「欸欸欸不行，那你怎麼辦——」

佐伯悠：「我家近。真的。」

（她盯著我看了兩秒，忽然噗地笑出來。）

小早川千夏：「佐伯同學，你這個人，嘴上冷冰冰的。」

佐伯悠：「……哪有。」

小早川千夏：「那這把黃傘我可扣住了，明天不還你，當人質。」

《B：一起走》

佐伯悠：「一把傘擠一擠。你轉車的巷口，順路。」

小早川千夏：「真的假的？那我可不客氣囉！」

（她一把撐開我的黃傘，動作大得差點戳到簷角，笑得毫無防備。）

小早川千夏：「走走走，末班車就交給命運，我們用跑的！」

佐伯悠：「傘下不要跑——」

（話還沒說完，她已經拽著我衝進雨裡。傘歪了，兩個人的肩膀都濕了半邊，她卻笑得比誰都開心。）

【站前小巷（雨中・暖黃燈下）】

（小巷的石板被雨洗得發亮，兩旁人家的窗透出暖光。轉角那台老販賣機嗡嗡地亮著。）

小早川千夏：「請你喝罐熱的，當謝禮。」

（她投了幣，「哐」一聲滾出兩罐溫熱的可可，塞一罐到我手裡。）

小早川千夏：「梅雨很煩對吧。悶、濕、鞋子永遠乾不了。」

佐伯悠：「嗯。」

小早川千夏：「可是……像今天這樣，反而會記很久欸。」

（她仰頭望著雨簾裡的暖黃燈，睫毛上還掛著水珠。）

小早川千夏：「明天要是還下雨，你美術教室畫完，還會趕末班嗎？」

佐伯悠：「……大概會。」

小早川千夏：「那就說好了。」

（她把空罐投進回收口，回頭朝我比了個手勢，像賽前起跑那樣。）

小早川千夏：「末班車集合。遲到的人，請熱可可。」

（遠處，末班車的燈光終於劃破雨幕，緩緩靠站。）

（我握著那罐還溫著的可可，忽然覺得，這誤點的五分鐘，賺到了。）
```
