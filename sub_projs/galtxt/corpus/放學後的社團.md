# 放學後的社團

放學後的社辦與文化祭——文藝社、趕工、顧攤。

## 放學後的文藝社

> **場景**：放學後的文藝社社辦，只有主角和一個女社員留下來
> **角色**：主角（男）、宮澤柚葉（女社員，逗趣傲嬌型）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（放學後的文藝社社辦。夕陽透過老舊的木窗，在地板上拉出長長的光影。）
宮澤柚葉：「喂，主角，你今天又在發呆了。」
主角：「哪有。我只是在看窗外那些快掉光了的櫻花。」
宮澤柚葉：「是嗎？我還以為你是在想等一下去『陽溜亭』要點薑汁燒肉還是可樂餅。」
（宮澤柚葉一邊說著，一邊把厚重的精裝書翻過一頁。）
主角：「被妳看穿了啊……話說回來，今天商店街那邊好像特別吵鬧呢。」
宮澤柚葉：「因為今天是『河鹿堂』進新書的日子吧，聽說二手書區都在打折。」
主角：「這樣啊，那等一下要順路去看看嗎？」
宮澤柚葉：「嗯……如果你肯幫我提書包的話，我可以勉為其難陪你去。」
主角：「明明是妳自己想買吧，每次都拿我當藉口。」
宮澤柚葉：「因為杜坂的斜坡很陡嘛，女生的體力可是很珍貴的。」
（她微微合上書本，指尖輕輕撫過書脊，轉頭看向窗外。）
宮澤柚葉：「不過，從這裡看下去的霞川，今天真的很漂亮。」
主角：「是啊。櫻花掉在河面上，看起來像一條粉紅色的帶子。」
宮澤柚葉：「……原來你真的有在看櫻花啊。」
主角：「所以就說了我沒有在想吃的了。」
宮澤柚葉：「好吧，相信你一次。那，我們走吧？」
（她站起身，拍了拍百褶裙上的裙擺，嘴角帶著一絲若有似無的笑意。）
```

## 文化祭前的社辦

> **場景**：文化祭前一週，放學後社辦趕工做看板/道具，忙亂，兩人邊做邊鬥嘴
> **角色**：主角（男）、清水遙（女社員，畫看板）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（夕陽照進凌亂的社辦，地板上散落著硬紙板與顏料罐，空氣中瀰漫著木屑的味道。）
清水遙：「喂，主角，把那邊的藍色壓克力顏料遞給我，快點。」
主角：「給。話說回來，妳這塊背景板上的天空，藍得有點像暴風雨前的海水啊。」
清水遙：「這叫『深夜的群青』！不懂藝術就不要亂指點，好好割你的瓦楞紙。」
主角：「我割得很認真好嗎？要是線歪了，某人明天又要一邊吃定食一邊跟我抱怨了。」
清水遙：「我才沒有吃著『陽溜亭』的炸豬排一邊抱怨……只是單純的意見交流。」
（清水遙接過顏料，用畫筆沾了沾，鼻尖不知何時沾上了一點白色的顏料。）
主角：「是是是，意見交流。不過，妳鼻子上有東西。」
清水遙：「騙人，你又想趁機搶我的水彩筆對吧？」
主角：「誰要搶那個。是真的，右邊鼻翼，像沾了糖霜一樣。」
（清水遙有些慌亂地用手背擦了擦，結果把白色顏料抹得更開了。）
清水遙：「……乾淨了嗎？」
主角：「完全沒有，現在看起來像長了白鬍子的貓。」
清水遙：「可惡，你別笑！快點幫我拿濕紙巾啦！」
主角：「好好，請用。不過，其實這樣也挺適合妳的。」
清水遙：「……你這傢伙，果然是在捉弄我吧？」
主角：「真的沒有，快畫吧，不然今天就趕不上回車站的末班車了。」
清水遙：「知道啦，等一下回去，你要請我喝果汁補償。」
主角：「好，買商店街那家轉角販賣機的。」
```

> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

