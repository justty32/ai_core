## 城鎮（日和町）

| 地點 | 目前已知的細節（從台詞累積） |
|------|------|
| **日和町（ひより）** | 內陸河谷小城，人口不多，「走十分鐘就會遇到認識的人」。夜裡安靜，聽得見電車。 |
| **霞川** | 橫貫鎮中央的河。兩岸櫻並木＋河堤步道。黃昏水色溫暖、橋墩處落櫻聚成一片。 |
| **商店街** | 有點沒落、傍晚仍熱鬧。石板路。轉角有販賣機（果汁、蜜柑汁）。 |
| **車站** | 老式木造月台，站房入夜亮暖黃燈。**末班車很早**（錯過只能走杜坂）。 |
| **河鹿堂** | 老書店。安靜，木百葉窗。文庫本、二手書打折區、有固定的「進新書日」。 |
| **陽溜亭** | 街角定食屋。招牌：薑汁燒肉、炸豬排、可樂餅、日替定食、冰淇淋汽水。 |

## 學校（縣立‧日和高等學校）

| 地點 | 目前已知的細節 |
|------|------|
| **杜坂** | 通學長坡，很陡。冬天薄雪易打滑；體育耐力跑繞它跑。 |
| **校門玄關** | 有鞋櫃（換室內鞋／製服皮鞋）。雨天在此躲雨。 |
| **教室** | 老校舍，木造地板一走就響。清晨無人時特別安靜；放學後斜陽。 |
| **頂樓天台** | 鐵門沒上鎖（風一吹就開），半默許的祕密基地。風大，看得到霞川。 |
| **圖書室** | 空調常壞，夏天只有一台舊電風扇。 |
| **社辦** | 文藝社／美術社。放學後趕工做看板道具（瓦楞紙、壓克力顏料）。 |
