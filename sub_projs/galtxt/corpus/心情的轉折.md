# 心情的轉折

情感的起伏——河堤黃昏的心緒、天台的和好、病後的關心。

## 黃昏的河堤

> **場景**：放學後的霞川河堤，黃昏，兩人繞路沿河走，看水看橋、看散落的櫻花
> **角色**：主角（男）、野上（女同學，家裡開定食屋）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
(黃昏的霞川河堤，風吹過落櫻，川面閃爍著橙黃的光芒。)
主角：「今天特地繞遠路，野上妳不急著回家嗎？」
野上：「嗯，家裡的定食屋今天休息，不用幫忙。」
主角：「這樣啊。不過，今天河水的顏色看起來特別溫暖。」
野上：「因為是黃昏嘛。你看，橋下的櫻花都聚在一起了。」
(她停下腳步，指向前方石橋的橋墩處，水流在那裡打轉。)
主角：「真的，像是一片粉紅色的地毯。」
野上：「總覺得，櫻花一落光，春天就真的要結束了。」
主角：「明明新學期才剛開始沒多久。」
(一陣微風吹過，幾片花瓣落在她的肩膀上，主角伸手幫她輕輕拂去。)
野上：「謝謝……感覺有點癢癢的。」
主角：「杜坂那邊的櫻花，大概明後天也會掉光吧。」
野上：「不過，綠芽長出來的時候也很好看，很有生機。」
主角：「確實是這樣。」
野上：「明天開始就是連假了，你打算做什麼？」
主角：「應該會去老車站那邊的河鹿堂看書吧。妳呢？」
野上：「那，我也去那裡找找新刊好了。明天見？」
主角：「嗯，明天見。」
```

## 天台的和好

> **場景**：兩人因小事鬧彆扭冷戰幾天，放學後在天台，一方主動來和好；先尷尬後緩和
> **角色**：主角（男）、吉岡春菜（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（老舊的天台鐵門被輕輕推開，發出吱呀的聲響）
春菜：「那個……」
（主角轉過身，有些驚訝地看著站在門口的春菜）
主角：「吉岡？妳怎麼……」
春菜：「看到你往天台走，就不自覺跟上來了。」
主角：「這樣啊。」
（春菜走到護欄旁，在距離主角兩步遠的地方停下，看著遠處的霞川）
春菜：「這幾天，你是不是都在躲著我？」
主角：「沒有的事。只是……不知道該用什麼表情面對妳。」
春菜：「笨蛋，明明是你先不理人的。」
主角：「明明是吉岡在陽溜亭先轉過頭去的吧？」
春菜：「因為你竟然說商店街的可樂餅比較好吃，明明是陽溜亭的更厲害。」
主角：「好吧，那是我嘴硬。陽溜亭的才是第一名。」
春菜：「……其實，商店街那家我也挺喜歡的。」
主角：「那我們這算是和好了？」
春菜：「嗯。這幾天，對不起。」
主角：「我也是。沒有妳在旁邊吵，放學的杜坂走起來挺寂寞的。」
（春菜有些害羞地鼓起腮幫子，隨即笑了出來）
春菜：「誰吵了啊……那，今天要去河鹿堂嗎？」
主角：「去啊，順便去陽溜亭吃晚餐吧。」
```

## 病後的筆記

> **場景**：主角感冒請假幾天後回校，女同學把整理好的筆記、講義帶來給他
> **角色**：主角（男）、長谷川（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（放學後的舊校舍教室，斜陽穿過窗戶，照在木造地板上）
主角：「……呼，終於把大掃除的抹布洗完了。」
長谷川：「辛苦了。感冒剛好就值日，真的沒問題嗎？」
主角：「嗯，已經完全退燒了，謝謝長谷川同學。」
（長谷川從書包裡拿出一疊整理好的筆記和講義，遞了過來）
長谷川：「這個，是這幾天課堂上的筆記，還有發下來的講義。」
主角：「哎？妳特地幫我整理的嗎？」
長谷川：「因為導師說，要是你回來了，就順便交給你。」
主角：「這樣啊……不過，這字跡整理得真整齊，幫了大忙。」
長谷川：「沒什麼，只是上課順便多寫了一份而已。」
主角：「英文課那疊，還用顏色標註了重點呢。」
長谷川：「……那是因為，下週就要小考了。」
主角：「哈哈，也是。對了，今天回去的路上，我請妳去陽溜亭吃可樂餅吧？」
長谷川：「哎？不用這麼客氣的……」
主角：「不行，這可是救命之恩。而且我也肚子餓了。」
（長谷川微微低下頭，輕輕把講義推到主角懷裡）
長谷川：「……那，我就不客氣了。不過，可樂餅要剛炸好的才行。」
主角：「沒問題，那就走吧。」
```
