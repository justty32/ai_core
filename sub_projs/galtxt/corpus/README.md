# corpus — galgame 台詞語料庫（舞台：日和町）

galtxt 的語料庫：以虛構小城「**日和町**」為舞台的日式 galgame 台詞、劇本、世界設定與分析。
用途＝idea 礦脈 §4.3「語料錨」——few-shot 黃金標準、reliability 訊號、有 ground truth 的 benchmark、蒸餾成規則的原料。基調：輕小說青春日常、**嚴格不油膩**、情感含蓄。

台詞格式：場景標題用 `【】`、旁白／心聲用 `（）` 另起行、對白用 `角色：「台詞」` 一行一句，整段台詞／劇本本體包在 ```` ``` ```` code block。

## Canon 專名（量產後收斂、以此為準）

- **縣**＝奧津縣（虛構內陸縣）。**町**＝日和町（河谷小城，霞川橫貫，主橋日和橋，南岸住宅區＝川原町）。
- **商店街**＝日和銀座；**定食屋**＝陽溜亭（野上家）；**書店**＝河鹿堂（蓋梅花章、耳背老爺爺、後有私藏書庫）；**米店**＝佐佐木家。
- **鐵道**＝霞川線（單線非電氣化支線，末班車約八點半＝「灰姑娘鐘」）；老式木造車站。
- **學校**＝縣立日和高等學校（山丘上，通學坡杜坂，老木造校舍，天台沒上鎖，森林綠西裝制服，社團鬆散，文化祭在秋）。**中學**＝日和第一中學校。
- **主角**＝佐伯悠（男，高一，選修美術，川原町住，愛上天台發呆，嘴硬心軟）。
- **女主（首篇劇本）**＝柏木秋穗（新生代表、日和町之光、河鹿堂常客）。
