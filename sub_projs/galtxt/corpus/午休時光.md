# 午休時光

午休前後——天台、便當、悶熱圖書室的片刻。

## 午休的天台

> **場景**：午休的頂樓天台，風有點大，主角上來透氣，發現女同學已經在那裡
> **角色**：主角（男）、森川（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（推開鐵門，強風迎面吹來，夾雜著幾片殘櫻）
主角：「啊，不好意思，我不知道已經有人在這邊了。」
森川：「沒關係的。這扇門本來就沒鎖，誰都可以進來。」
（風又大了一點，少女伸手按住耳邊被吹散的髮絲）
主角：「今天風真的很大呢，總覺得再吹一下，櫻花就全落光了。」
森川：「嗯。從這裡看下去，霞川兩岸的櫻木已經開始變綠了。」
主角：「是啊，早上走杜坂上學的時候，斜坡上全都是落花。」
森川：「總覺得有點可惜。不過，新學期大概就是這樣的感覺吧。」
主角：「新學期的感覺？」
森川：「就是……大家都很有精神地迎向新開始，只有自己還留在原地之類的。」
主角：「原來森川同學也會想這些啊。」
森川：「我也只是個普通的高中生呀。因為教室裡太吵了，所以才想來這裡透透氣。」
主角：「那，我是不是打擾到你了？」
森川：「沒有的事。兩個人在這裡的話，風吹起來好像也沒那麼冷了。」
主角：「是嗎……那我可以在這裡待到午休結束嗎？」
森川：「請便。不過，待會要是感冒了可別怪我喔。」
```

## 午休的便當

> **場景**：午休的教室，主角忘了帶便當，女同學分便當給他
> **角色**：主角（男）、長谷川千穗（女同學，坐前排）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（午休的教室裡，陽光透過窗戶灑在課桌上，外頭杜坂的櫻花正悄悄飄落）
（主角盯著空無一物的桌面，無奈地嘆了口氣）
主角：「今天早上真的太匆忙了……」
（坐在前排的長谷川千穗轉過身來，手裡抱著一個雙層的木質便當盒）
千穗：「主角君，難道是忘記帶便當了嗎？」
主角：「嗯，出門時放在玄關的鞋櫃上，完全忘記拿了。」
千穗：「真是的，這已經是這學期第二次了吧。」
（千穗把自己的便當盒放在主角的桌上，輕輕揭開木蓋）
千穗：「不介意的話，這個分你一半吧。」
主角：「這怎麼行，那長谷川自己不就不夠吃了？」
千穗：「不用擔心，今天媽媽多裝了一些，而且……我也吃不了那麼多。」
（千穗用便當盒的蓋子分出一半的飯菜，推到主角面前，遞過去一雙木筷）
主角：「那……我就不客氣了。謝謝妳，長谷川。」
千穗：「嗯，請用吧。對了，那個玉子燒是甜的喔。」
（主角夾起一塊玉子燒放進嘴裡）
主角：「真的，甜甜的，非常美味。」
千穗：「太好了。因為我比較喜歡甜的，還怕主角君吃不習慣呢。」
主角：「完全不會，這味道剛剛好。下次我一定會帶謝禮來的。」
（千穗微微低下頭，用筷子輕輕戳著便當盒裡的配菜）
千穗：「謝禮什麼的就不用了……只要明天主角君記得帶便當就好。」
主角：「哈哈，那明天出門前我一定先檢查三遍。」
千穗：「嗯，那就約好了喔。」
```

## 盛夏的圖書室

> **場景**：盛夏午後、悶熱的圖書室，只有電風扇聲，兩人各自唸書偶爾閒聊
> **角色**：主角（男）、吉岡（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（老舊的電風扇在角落發出規律的運轉聲，吹出溫熱的風。）
吉岡：「主角，你那題數學算出來了嗎？」
主角：「還沒，三角函數的公式總覺得記不太熟。」
（吉岡用自動鉛筆的筆帽輕輕敲了敲額頭。）
吉岡：「我也是。而且……好熱啊，感覺腦袋都快融化了。」
主角：「再撐一下吧，圖書管理員說空調壞了，今天只能靠這台電扇。」
吉岡：「明明去年夏天也是這樣，學校到底什麼時候才要修理啊。」
（她把微濕的雙頰邊的頭髮撥到耳後。）
主角：「可能預算都拿去整修杜坂的通學路了吧。」
吉岡：「也是呢。啊，這個給你。」
（吉岡從口袋裡摸出一顆藍色包裝的糖果，悄悄推到課本中間。）
主角：「這是……汽水糖？」
吉岡：「嗯，鹽分補給。聽說這樣比較不會中暑。」
主角：「謝謝。妳不吃嗎？」
吉岡：「我剛才在走廊上已經吃過一顆了。」
主角：「好冰……感覺稍微活過來了。」
吉岡：「對吧？那，算完這頁的話，等一下去陽溜亭吃冰淇淋汽水吧？」
主角：「好啊，我請客。」
吉岡：「一言為定。那我要繼續寫囉。」
```
