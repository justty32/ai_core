# 教室的日常

日和町・縣立日和高等學校，發生在教室裡的日常場景。

## 清晨的教室

> **場景**：清晨、還沒什麼人的教室，主角比平常早到，遇到早到值日的女同學
> **角色**：主角（男）、秋山小春（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（清晨的教室，陽光穿過靠窗的玻璃，照亮空氣中細小的灰塵。）
（秋山小春坐在倒數第二排靠窗的位子，手裡拿著抹布，正靜靜地看著窗外。）
主角：「……早。」
（小春有些驚訝地轉過頭，隨即微微一笑。）
秋山小春：「啊，早安。今天真早呢。」
主角：「嗯，路上沒什麼人，杜坂的櫻花也開始散了，就不自覺走得快了點。」
秋山小春：「是呢，櫻花地毯鋪了一整街。剛才上來的時候，我差點捨不得踩下去。」
主角：「妳在擦窗台嗎？」
秋山小春：「沒有，只是剛好輪到值日生，想說早點來做完。」
主角：「那……我也來幫忙吧。」
（主角走到教室後方的櫃子旁，拿了另一塊抹布。）
秋山小春：「謝謝。那，可以拜託你擦後門那邊的置物櫃嗎？」
主角：「沒問題。」
（兩人各自安靜地擦拭著。老舊的木造地板隨著腳步發出輕微的咯吱聲。）
主角：「今天早上的風，好像還有一點涼。」
秋山小春：「嗯，不過陽光曬到的地方已經很暖和了。」
（小春停下手上的動作，看著窗外遠處在晨光下波光粼粼的霞川。）
秋山小春：「新學期才剛開始，總覺得，早上的教室讓人心情特別平靜。」
主角：「是啊，等大家都到了，就會變得很吵了。」
秋山小春：「那樣也很好，不過……我稍微更喜歡現在這個時間一點。」
主角：「我也是。」
```

## 放學後的教室

> **場景**：放學後的教室，四月新學期第一週，櫻花將散
> **角色**：主角（男）、遠藤葵（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（放學後的教室。夕陽將課桌椅拉出長長的影子，窗外傳來運動社團隱約的呼喊聲。）
主角：「遠藤，還不走嗎？值日生的工作已經做完了吧。」
遠藤：「再等一下。這頁的筆記還沒抄完。」
（遠藤葵一邊說著，手中的原子筆一邊在課本上發出沙沙的聲響。）
主角：「真是認真啊。明明才開學第一週而已。」
遠藤：「因為落後的話，之後會很麻煩吧。……而且，杜坂的坡道很陡，我想等太陽再下山一點再走。」
主角：「哈哈，這倒是。每天爬那條通學坡簡直是體能訓練。」
（主角拉過前排的椅子，反過來坐下，趴在靠背上望著窗外。）
遠藤：「你看，從這裡可以看到霞川的櫻花呢。雖然已經開始散落了。」
主角：「嗯，從櫻並木那裡吹過來的風，好像還帶著一點花香。」
遠藤：「你今年選修了美術課對吧？」
主角：「對啊，因為聽說比較輕鬆。怎麼了？」
遠藤：「沒什麼。只是想著，如果是你的話，應該能把這幅景色畫得很好。」
（遠藤停下筆，微微側過頭，看著主角。）
主角：「我也只是亂畫而已。比起這個，你的筆記寫完了嗎？」
遠藤：「……嗯，寫完了。」
（她輕輕合上筆記本，發出清脆的聲響。）
主角：「那就走吧。正好我也休息夠了。」
遠藤：「嗯。今天走河堤那邊吧，我想再近距離看一下櫻花。」
主角：「好啊，順便去車站前買個可樂餅。」
遠藤：「那，我要紅豆餅。」
```

## 考前的教室

> **場景**：期中考前，放學後的教室，兩人留下來一起唸書、互相考對方、抱怨考試
> **角色**：主角（男）、吉野真衣（女同學）
> **出處**：agy 生成（Gemini 3.5 Flash）＋人工審過合背景

---

```
（落日餘暉斜照進放學後的教室，只剩下兩張併在一起的課桌）
吉野真衣：「啊……不行了，古典文學的助動詞變化到底是誰發明的啊。」
主角：「抱怨也沒用，後天就要考了。不然，我隨便考妳一個？」
吉野真衣：「來吧，我準備好了！」
主角：「如果句尾是『ぞ』，後面的動詞要變成什麼形？」
（吉野用自動鉛筆的筆帽輕輕抵著下巴，苦惱地盯著黑板）
吉野真衣：「唔……是變成連體形……對吧？」
主角：「答對了。換妳考我。」
吉野真衣：「聽好了，大化革新的年份是？」
主角：「六百四十五年。這題根本是送分題吧。」
吉野真衣：「可惡，為什麼你答得這麼順暢。真不公平。」
主角：「因為我在上課時沒有偷偷看窗外的櫻花。」
吉野真衣：「那、那是因為今年櫻花散落的樣子特別好看嘛……」
（她有些不好意思地移開視線，用手指輕輕旋轉著橡皮擦）
主角：「是是是。但如果妳數學再不及格，就真的要跟陽溜亭的薑汁燒肉說再見了。」
吉野真衣：「嗚，被我媽禁食一個月的話，我真的會撐不下去的。」
主角：「那我晚上把整理好的公式筆記傳給妳。所以，現在把這題算完吧。」
吉野真衣：「真的？那考完試，我請你喝老車站前那台販賣機的蜜柑汁。」
```
