# 陽溜亭・店誌

> 街角的定食屋。既有設定既定的招牌：**薑汁燒肉、炸豬排、可樂餅、日替定食（如紅豆湯圓）、冰淇淋汽水**。這裡把它深挖成一間可坐下來吃飯的店。
> 標記：**【定】**＝既有設定既定；**【推】**＝為一致性補的設定，可改。

---

## 一、店名與位置

- **陽溜亭（ひだまりてい）**【定】。「陽溜まり」＝向陽處、暖洋洋的一小塊陽光——店名即氣氛。
- 位在**商店街轉角**【定，街角的定食屋】。油鍋的香味飄很遠，放學路上聞得到炸豬排／薑汁燒肉的味道【定，見城鎮的傍晚語料「陽溜亭飄來的炸豬排香味」】。
- 老式食堂的門面：暖簾、玻璃拉門、櫥窗裡褪色的**食品模型（蠟樣本）**、手寫的紙短冊菜單貼滿牆【推】。

---

## 二、店內【推】

- 進門一台老收銀機、擺著牙籤與醬油瓶的桌。
- 座位：靠牆幾張四人桌、窗邊卡座、還有面對廚房的一排吧台位（一個人來吃飯坐這）。
- 桌上：醬油、七味、爪楊枝、免費的冷麥茶（夏）／熱焙茶（冬）。
- 一台老電視掛在角落，播著相撲或地方新聞；牆上月曆是米店或酒舖送的。
- 廚房半開放，看得見鐵板與炸鍋的煙、聽得見「入りました！」的吆喝。
- **氣味**：醬汁、炸油、白飯的熱氣【定，見佐佐木語料「熱騰騰的煙霧伴隨醬汁香氣」】。
- **光**：如其名，午後有一塊陽光正落在窗邊卡座——「陽溜まり」的位子，搶手。

---

## 三、店主一家【推】

- **老闆（大將）**：掌鐵板與炸鍋，話不多、手快。薑汁燒肉的醬汁配方是他的驕傲【呼應佐佐木語料「陽溜亭的薑汁燒肉果然是最棒的」】。
- **老闆娘（女將）**：外場招呼、記單、跟熟客與學生閒聊；會多添半碗飯給看起來餓的學生。到立花青果店、中村米店、相川鮮魚店進貨的是她【呼應商店街店家名冊.md】。
- **看板娘／幫手**（可留白）：或有個上高中／打工的女兒或姪女幫忙端盤——先不定死，等台詞需要再補【推，比照背景設定「不先立人設表」】。

## 四、菜單全表

> 招牌四樣為【定】，其餘為【推】補的合理定食屋菜單。價位是「學生下得了手」的地方食堂價。

### 招牌・定番【定】
| 品項 | 說明 |
|------|------|
| **薑汁燒肉定食（生姜焼き）** | 招牌中的招牌。醬汁是大將的驕傲。附白飯、味噌湯、高麗菜絲、漬物。 |
| **炸豬排定食（とんかつ）** | 現炸厚切，香味飄滿商店街。附豬排醬、芥末、高麗菜絲。 |
| **可樂餅（コロッケ）** | 可單點、可做定食。外酥內鬆。與商店街三好精肉店的炸物同屬本鎮靈魂食物【呼應物產與名物.md、商店街店家名冊.md】。 |
| **日替定食（日替わり）** | 每天換的今日特餐【定】。內容看大將心情與當季進貨（見下方輪替表）。 |
| **冰淇淋汽水（クリームソーダ／アイスクリームソーダ）** | 招牌飲品／甜點【定】。汽水上浮一球冰淇淋，玻璃杯、長匙、紅櫻桃。夏天必點，小孩眼睛發亮。 |

### 定食・丼類【推】
- 炸雞塊定食（唐揚げ）、炸蝦定食、鯖魚鹽燒定食、薑燒改雞的「雞肉薑燒」。
- 親子丼、豬排丼（カツ丼）、天丼；咖哩飯（小孩最愛）。

### 麵類【推】
- 拉麵（醬油）、烏龍、蕎麥（冬天手打蕎麥，扣宿場遺風）【呼應物產與名物.md】、夏天冷やし中華。

### 單品・小菜【推】
- 可樂餅、炸火腿排、玉子燒、冷奴、漬物盤、味噌湯。

### 甜點・飲物【推＋定】
- **冰淇淋汽水**【定】、**紅豆湯圓（ぜんざい／白玉）**【定，日替甜點之一】、餡蜜、刨冰（夏）、汁粉（冬）。
- 瓶裝彈珠汽水、冷麥茶、熱焙茶。

## 五、日替定食・一週輪替（示意）【推】

> 日替是「當天限定」的樂趣，實際看大將與當季進貨。**紅豆湯圓／冰淇淋汽水**是既定會出現的品項【定】。

| 曜日 | 日替（示意） |
|------|------|
| 月 | 薑汁燒肉的兄弟版・味噌豬肉定食 |
| 火 | 炸雞塊定食＋小碗紅豆湯圓（ぜんざい） |
| 水 | 鯖魚鹽燒定食 |
| 木 | 可樂餅＋炸火腿排的「炸物拼盤定食」 |
| 金 | 咖哩豬排（週五犒賞自己） |
| 季節 | 夏＝鹽烤香魚定食＋冰淇淋汽水；冬＝手打蕎麥＋汁粉；秋＝舞菇炊飯 |

*註：日替甜點常見**紅豆湯圓**與**冰淇淋汽水**——這兩樣是店的定番，日替只是換主菜。*

## 六、常客與這家店的角色【推＋定】

- **日和高中生**：放學來吃、比學校食堂的麵包好太多【定，見佐佐木語料】；體育課跑完杜坂餓扁了來補一碗【呼應語料】。
- 一個人來吃飯的上班族、下工的大叔、川原町的老人。
- 與**河鹿堂**是一對：一個管肚子、一個管心；「陽溜亭吃飯、河鹿堂看書」是本鎮放學後的黃金動線【呼應城鎮的傍晚語料、商店街】。
- 對本鎮：陽溜亭是沒落商店街裡**最有生氣、最暖**的一角——燈亮、油香、人聲、熱飯，是「傍晚商店街還算熱鬧」那句話的具體來源【定】。

---

## 七、一句話定位

> 街角轉彎、油香先到、暖簾一掀熱氣撲面的定食屋。薑汁燒肉是大將的驕傲，冰淇淋汽水是小孩的節日，日替定食是這座鎮每天的小小驚喜。放學後，先來這裡填飽肚子。

*陽溜亭的硬設定（街角、薑汁燒肉／炸豬排／可樂餅／日替／冰淇淋汽水、飄香的油鍋）一律保留；這裡補的是坐下來的細節。看板娘等人物先留白，比照全書「不先立人設表」的原則。*
