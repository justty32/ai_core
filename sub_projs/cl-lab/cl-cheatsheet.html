02 資料結構

容器建立取值

list

(first '(1 2 3)) (rest '(1 2 3))  ;1  (2 3)
(cons 0 '(1 2))              ;=>(0 1 2)
(mapcar #'1+ '(1 2 3))       ;=>(2 3 4)
(remove-if-not #'evenp '(1 2 3 4)) ;=>(2 4) filter

vector（可長大）

(aref #(10 20 30) 1)         ;=>20
(defparameter v (make-array 0 :adjustable t :fill-pointer 0))
(vector-push-extend 99 v)        ; v => #(99)

hash-table

(defparameter h (make-hash-table :test 'equal))
(setf (gethash "k" h) "v")      ; 寫入
(gethash "k" h)               ;=> "v", t
(maphash (lambda (k v) ...) h) (remhash "k" h)

字串 key 一定要 :test 'equal。gethash 回兩值：值＋有沒有找到。

03 JSON · jzon

(json:stringify #(1 2 3))          ;=> "[1,2,3]"
(json:stringify h :pretty t)      ; 縮排

CLJSON
hash-table(equal,字串key)object
list / vector（非空）array
t / niltrue / false
nil 陷阱：nil→false（不是空陣列）。空陣列用 #()、空物件用空 hash-table。取 JSON array 元素用 (aref v i)。

04 CLI · clingon

(clingon:make-command
  :name "greet" :description "..."
  :options
  (list (clingon:make-option :list :description "名字"
          :long-name "name" :short-name #\n :key :name)
        (clingon:make-option :flag :description "大寫"
          :long-name "upper" :short-name #\u :key :upper))
  :handler #'h)
(clingon:run (cmd))   ; 讀真正 argv、跑完會 exit

handler 內：(clingon:getopt cmd :name) 取選項、(clingon:command-arguments cmd) 取位置參數。

kind取出
:string / :integer值
:flagt/nil
:list可重複→串列
:counter-vvv→次數

子命令：頂層 :sub-commands (list (add-cmd) ...)，每個各有 --help，可任意巢狀（git submodule 風格）。make-option 也必填 :description。

08 條件系統 CL 靈魂

; try/catch
(handler-case (error "x")
  (error (e) (format nil "抓到 ~a" e)))
; 出錯回 nil (+條件)
(ignore-errors (error "x"))    ;=> nil, #<error>
; finally
(unwind-protect (work) (cleanup))

restart：發生與處理分離

(defun safe-div (a b)
  (restart-case (if (zerop b) (error "/0") (/ a b))
    (use-zero () 0)))          ; 提供復原方案
(handler-bind ((error (lambda (c)
                (invoke-restart 'use-zero))))
  (safe-div 1 0))                 ;=> 0（從錯誤點繼續）

高層決策(handler-bind)與低層復原(restart-case)解耦——這就是 SBCL debugger 的底層。

07 macro

; ` 模板  , 取值  ,@ 攤平串列
(defmacro my-when (test &body body)
  `(if ,test (progn ,@body)))

(defmacro swap (a b)
  (let ((tmp (gensym)))          ; 避免變數捕捉
    `(let ((,tmp ,a)) (setf ,a ,b ,b ,tmp))))

(macroexpand-1 '(my-when x (foo)))  ; 除錯關鍵
;=> (IF X (PROGN (FOO)))

函式處理值、macro 處理未求值的程式碼。要控制求值/新語法才用 macro，其餘用函式。

09 C FFI · cffi

(ql:quickload :cffi)
(cffi:load-foreign-library "libm.so.6")
(cffi:defcfun ("sqrt" c-sqrt) :double (x :double))
(c-sqrt 2.0d0)                   ;=> 1.4142...
(cffi:defcfun ("strlen" c-strlen) :unsigned-long (s :string))
(c-strlen "hello")              ;=> 5

型別：:int :long :double :float :string(char*) :pointer :void :bool。浮點用 2.0d0(double) 對上 C。

⚠ 地雷合輯

症狀原因 / 正解
sbcl --script 報 "Package QL does not exist"--script 不讀 ~/.sbclrc，Quicklisp 沒載。腳本開頭 (load .../quicklisp/setup.lisp)。
把 quickload 和用到該套件符號的碼包在同一 form → 讀取錯誤整塊在讀取期就解析、套件還沒載。quickload 要獨立成前面的 top-level form。
JSON 布林欄位塞 nil 卻變 false；空陣列變 falsejzon：nil→false。空陣列用 #()、空物件用空 hash-table。
字串當 hash-table key 卻取不到預設 eql 不比字串內容。(make-hash-table :test 'equal)。
取 JSON array 元素用 nth 失敗jzon parse array → vector 不是 list。用 (aref v i)、loop for x across v。
clingon 報 "Must specify description"每個 make-option 都必填 :description（不只 command）。
FFI 傳 2.0 給 :double 拿到垃圾用 double 字面 2.0d0；型別要對上 C。
