# 02 · 資料結構（list / vector / hash-table）

CL 的三大容器。心法：

| 容器 | 建立 | 取值 | 適合 |
|------|------|------|------|
| **list** 串列 | `'(1 2 3)` / `(list ...)` | `(nth i l)` / `first` | 小量、順序處理、程式碼即資料 |
| **vector** 向量 | `#(1 2 3)` / `make-array` | `(aref v i)` O(1) | 需索引、大量、可長大 |
| **hash-table** 雜湊表 | `make-hash-table` | `(gethash k h)` | key→value 對應 |

## 串列 list

```lisp
(list 1 2 3)                 ; => (1 2 3)
'(1 2 3)                     ; 同上（quote，不求值內容）
(first '(1 2 3))            ; => 1     （= car）
(rest '(1 2 3))             ; => (2 3) （= cdr）
(nth 2 '(a b c))            ; => c     （O(n)，別拿 list 當隨機存取）
(cons 0 '(1 2))            ; => (0 1 2) 前插
(append '(1 2) '(3 4))     ; => (1 2 3 4)
(length '(1 2 3))          ; => 3
```

高階操作（回傳新串列，不改原本）：

```lisp
(mapcar (lambda (x) (* x x)) '(1 2 3 4))   ; => (1 4 9 16)
(remove-if #'evenp '(1 2 3 4 5))           ; => (1 3 5)   移掉符合的
(remove-if-not #'evenp '(1 2 3 4 5))       ; => (2 4)     只留符合的（= filter）
(reduce #'+ '(1 2 3 4))                    ; => 10
(find 3 '(1 2 3))                          ; => 3 / nil
(member 3 '(1 2 3))                        ; => (3)  找到回「該位置起的尾巴」
```

> `'(...)` 是**字面常數**，不要對它做破壞性修改（`nconc`/`setf (car ...)`）；要可變就 `(list ...)`。

## 向量 vector

```lisp
#(10 20 30)                                 ; 字面向量
(aref #(10 20 30) 1)                        ; => 20   O(1) 索引
(defparameter *v* (make-array 3 :initial-contents '(10 20 30)))
(setf (aref *v* 0) 99)                       ; 原地改
```

**可長大的向量**（像其它語言的動態陣列）：`:adjustable t :fill-pointer 0`

```lisp
(defparameter *dv* (make-array 0 :adjustable t :fill-pointer 0))
(vector-push-extend 99 *dv*)                 ; *dv* => #(99)
(length *dv*)                                ; => 1
```

字串就是字元向量，`(aref "abc" 0)` => `#\a`。

## 雜湊表 hash-table

```lisp
(defparameter *h* (make-hash-table :test 'equal))   ; key 用 equal 比較（字串 key 必須）
(setf (gethash "k" *h*) "v")                 ; 寫入（setf + gethash）
(gethash "k" *h*)                            ; => "v", t   （第二值：有沒有找到）
(gethash "no" *h* :default)                  ; => :default, nil
(hash-table-count *h*)                       ; => 1
(remhash "k" *h*)                            ; 刪 key
(maphash (lambda (k v) (format t "~a=~a~%" k v)) *h*)  ; 走訪
```

**`:test` 很重要**：預設 `eql`（適合數字/符號 key）；**字串 key 一定要用 `'equal`**，
否則兩個內容相同的字串會被當成不同 key。

`gethash` 回**兩個值**：值本身、以及「有沒有這個 key」（用來區分「值是 nil」和「key 不存在」）。
