# 02 · 資料結構（array / tuple / table / struct）

Janet 的集合分「可變」與「不可變」兩套，一個 `@` 就是差別：

| 概念 | 可變（前綴 `@`） | 不可變 | 你熟悉的對應 |
|------|------------------|--------|--------------|
| 陣列 | `@[1 2 3]` array | `[1 2 3]` tuple | list / array |
| 雜湊表 | `@{:a 1}` table | `{:a 1}` struct | hash map / dict |

規則：**要改就用 `@`（array / table），當常數就用不可變（tuple / struct）**。
不可變的 struct / tuple 可以當 table 的 key，可變的不行。

## 陣列 array `@[...]`

```janet
(def arr @[1 2 3])
(array/push arr 4)          # arr 變 @[1 2 3 4]（原地改）
(arr 0)                     # => 1        用索引當函式呼叫來取值
(length arr)                # => 4
(array/slice arr 1 3)       # => @[2 3]   取子陣列

(map |(* $ $) arr)          # => @[1 4 9 16]
(filter even? arr)          # => @[2 4]
(reduce + 0 arr)            # => 10
```

tuple 是不可變版，取值一樣 `(tup 2)`，但不能 push / 改。

## 雜湊表 table `@{...}`

```janet
(def t @{:name "Alice" :age 30})
(put t :city "Taipei")      # 原地新增 / 覆寫一個 key
(t :name)                   # => "Alice"   把 key 當函式呼叫來取
(get t :age)                # => 30        get 也行，key 不存在回 nil
(get t :xxx :fallback)      # => :fallback 第三參數是預設值
(has-key? t :city)          # => true
(update t :age inc)         # 用函式更新：age 30 -> 31
```

看整張表的 key / value / pair：

```janet
(keys t)                    # => @[:age :name :city]
(values t)                  # => @[30 "Alice" "Taipei"]
(pairs t)                   # => @[(:age 30) (:name "Alice") (:city "Taipei")]
```

**地雷**：取「巢狀」的值不要寫 `(t :a :b)`——第二個參數是**預設值**不是下一層 key。
要鑽多層用 `get-in`：

```janet
(def nested @{:user @{:names @["Zed"]}})
(get-in nested [:user :names 0])   # => "Zed"
(put-in nested [:user :age] 5)     # 深層寫入
```

## struct `{...}`（不可變 table）

```janet
(def s {:a 1 :b 2})
(s :a)                      # => 1
(merge {:a 1} {:b 2})       # => @{:a 1 :b 2}   合併成新 table（右邊蓋左邊）
```

`merge` 回傳的是 table（可變）。要不可變版用 `(struct ;(kvs ...))` 或 `table/to-struct`。

## 走訪

```janet
(each v @[10 20 30] (prin v " "))          # 值：10 20 30
(eachp [k v] @{:a 1 :b 2} (print k v))     # key-value 對
(seq [[k v] :pairs @{:a 1 :b 2}] [k v])    # 推導出 pair 陣列
```

## 小抄

| 想做 | 陣列 | 表 |
|------|------|-----|
| 建立 | `@[...]` | `@{...}` |
| 取值 | `(arr i)` | `(t k)` / `(get t k dflt)` |
| 新增 / 改 | `(array/push arr v)` | `(put t k v)` |
| 巢狀取 / 寫 | `(get-in a [i j])` | `(get-in t [k1 k2])` / `put-in` |
| 長度 | `(length arr)` | `(length t)` |
| 全部 key / value | — | `(keys t)` / `(values t)` |
| 走訪 | `(each v arr ...)` | `(eachp [k v] t ...)` |
