01 語言核心

綁定

(def x 10)          # 不可變
(var y 20) (set y 30)  # 可變
(let [a 1 b 2] (+ a b)) # 區域 => 3

函式

(defn add [a b] (+ a b))
(defn greet [&opt who]        # 可選參數
  (default who "world")
  (string "Hi, " who))
(defn sum [& xs] (reduce + 0 xs)) # 變長
(map |(* $ 2) @[1 2 3]) # |(..) $=首參

條件

(if (> x 5) "big" "small")
(when (> x 5) (print "big")) # 只真、多句
(cond (< x 0) "neg"  (= x 0) "zero"  "pos")
(case x 10 "ten" "other")

真假：只有 nil 與 false 是假，0／""／空集合都算真。

迴圈 / 推導

(loop [i :range [0 3]] (prin i)) # 副作用 012
(each v @[:a :b] (prin v))       # 走訪值
(seq [i :range [0 5] :when (odd? i)]
  (* i 10))  ;=> @[10 30]  推導回陣列

✦ 括號家族 () [] {} @

() 是「呼叫」，[]/{} 是「資料」。這是 Janet 跟傳統 Lisp 最不同、也是它 JSON 很順的根本。

(+ 1 2)           ;() 當呼叫 => 3
[1 2 3]          ;[] => tuple（不可變），內容會求值
[(+ 1 1) 2]     ;=> (2 2)
{:a (+ 40 2)}    ;{} => struct => {:a 42}
寫法型別可變
[1 2]tuple✗
@[1 2]array✓
{:a 1}struct✗
@{:a 1}table✓

{:a 1 :b [1 2 3]} 這種「像 JSON 的東西」就是合法 Janet 字面資料，可內含計算值。

02 資料結構

可變加 @：@[] array · @{} table；沒 @ 的 [] tuple · {} struct 不可變。

陣列 @[…]

(def a @[1 2 3])
(array/push a 4)      ;=> @[1 2 3 4] 原地
(a 0)                ;=> 1  索引當函式
(array/slice a 1 3) ;=> @[2 3]
(map |(* $ $) a)     ;=> @[1 4 9 16]
(filter even? a)     ;=> @[2 4]
(reduce + 0 a)      ;=> 10

雜湊表 @{…}

(def t @{:name "Alice" :age 30})
(put t :city "Taipei") # 原地寫
(t :name)              ;=> "Alice"
(get t :age :none)    # 第三參 = 預設
(has-key? t :city)    ;=> true
(update t :age inc)   # 用函式更新
(keys t) (values t) (pairs t)

巢狀 & 合併

(get-in n [:user :names 0]) # 鑽多層
(put-in n [:user :age] 5)
(merge {:a 1} {:b 2})     ;=> @{:a 1 :b 2}

03 JSON · spork/json

(import spork/json)
(def d @{:name "Bob" :n 3})

(json/encode d)          ;=> {"name":"Bob","n":3}
(json/encode d "  " "\n") # 縮排好讀版

(json/decode s)          # key 是「字串」
(json/decode s true)     # ★ key 轉 keyword（建議）

永遠傳 true，解回來就能 (d :name) 用 keyword 取值，跟自己寫的 Janet 一致。

檔案 round-trip

(spit "c.json" (json/encode d "  " "\n"))
(def back (json/decode (slurp "c.json") true))
JSONJanet
object / array@{} / @[]
string / number / bool同名
nullkeyword :null（decode 傳 3ⁿᵈ true → nil）

巢狀取 / 改

(get-in j [:a :b 1])          # 取（別用 (j :a :b)）
(update-in j [:a :b 1] |(* $ 100)) # 改
(map |($ :n) users)          # decode 出來就是普通 array/table
坑：開 nils=true 時值為 null 的鍵會整個消失（table 存不了 nil）。

04 CLI · spork/argparse

(import spork/argparse :as ap)
(def res
  (ap/argparse "說明"
    "name"  {:kind :accumulate :short "n"}
    "upper" {:kind :flag       :short "u"}
    "level" {:kind :option     :default "1"}
    :default {:kind :accumulate})) # 位置參數
(unless res (os/exit 1)) # nil = 失敗/印過help

給 -n Alice --name Bob --upper p1 p2：

(res "name")   ;=> @["Alice" "Bob"]
(res "upper")  ;=> true
(res "level")  ;=> "1"  (預設)
(res :default) ;=> @["p1" "p2"]
:kind意思型別
:flag開關bool
:multi可多次、數次數int
:option帶一值string
:accumulate帶值可多次array

:option/:accumulate 取回都是字串，要數字用 :map scan-number。--help 自動生成。

09 Fiber 協程

一段可暫停/恢復的執行流。generator、例外、非同步都是它。

generator

(def f (fiber/new (fn [] (yield 1) (yield 2))))
(resume f) (resume f)   ;=> 1 然後 2
# 直接走訪：
(each x (fiber/new (fn [] (for i 0 4 (yield (* i i)))))
  (prin x " "))       ;=> 0 1 4 9

錯誤處理（其實也是 fiber）

(try (error "boom") ([e] (string "抓到:" e)))
(protect (error "bad")) ;=> (false "bad")
(protect (+ 1 2))     ;=> (true 3)

ev：非同步（單執行緒協作）

(ev/spawn (print "背景"))  # 丟去背景跑
(ev/sleep 0)              # 讓排程器運轉
# 寫起來像同步、沒有 async 染色

08 巨集 macro

符號意思
~quasiquote：引用程式、可挖洞
,unquote：挖洞，這裡求值
,;unquote-splice：挖洞並攤平
'quote：純引用不求值
(defmacro my-when [c & body]
  ~(if ,c (do ,;body)))
(macex1 '(my-when x (foo))) # 看展開！
;=> (if x (do (foo)))

# 衛生：用 with-syms 生不撞名的臨時變數
(defmacro swap [a b]
  (with-syms [t]
    ~(let [,t ,a] (set ,a ,b) (set ,b ,t))))

能用函式就別用巨集；只在「要延遲求值/看未求值結構」時出手。

10 與 C 互通

方式編 C?用途

native 模組jpm 編用 C 寫 Janet 函式

FFI（不編譯，直呼）

(ffi/call (ffi/lookup m "cos") sig 0.0) ;=> 1.0

11 行程 / 管線 / 信號

(os/execute ["echo" "hi"] :p) # exit code；:p=用PATH :x=非零報錯
(import spork/sh)
(sh/exec-slurp "git" "rev-parse" "HEAD") # 抓輸出

管線

# 法一：Janet 中轉（好控制）
(def p (os/spawn ["sort"] :p {:in :pipe :out :pipe}))
(:write (p :in) "b\na\n") (:close (p :in))
(:read (p :out) :all) ;=> "a\nb"
# 法二：多段交給 shell
(os/execute ["sh" "-c" "a | b | c"] :p)
坑：別把 proc1 的 :out 直接接 proc2 的 :in——:pipe 是非阻塞 fd，對方讀會 EAGAIN。用上面兩法之一。

信號

(os/proc-kill child false :term) # signal 是 keyword
(os/sigaction :int (fn [] (print "bye")))
# ★ handler 要有 ev loop 在跑才會觸發

signal：:term :int :hup :kill :usr1 :usr2 :quit。

✦ 印 / 除錯

最常踩：print 對 array/table 只印位址 <array 0x…>，不是內容。
(pp @[1 2 3])        # pretty-print 內容
(printf "%q" x)      # 內嵌字串：可讀表示
(prin ...) (print ...) # 不換行 / 換行
(doc defn)          # REPL 查文件
%q/%p/%j/%n 都印 Janet 表示法（%j 是單行，不是 JSON）。真 JSON 只有 (print (json/encode x))。

⚠ 地雷合輯

症狀原因 / 正解
(print arr) 印出 <array 0x…>用 (pp arr) 或 (printf "%q" arr)；print 不格式化容器。
(t :a :b) 想取巢狀卻拿到怪值第二參數是預設值不是下一層 key。巢狀用 (get-in t [:a :b])。
JSON 解回來 (d :name) 拿到 nildecode 沒傳 true，key 是字串。傳 true 轉 keyword。
argparse --level 5 拿到 "5" 不能算術取回是字串。用 scan-number 或選項加 :map scan-number。
想改集合卻報錯忘了 @：不可變 []/{} 不能改，要 @[]/@{}。
