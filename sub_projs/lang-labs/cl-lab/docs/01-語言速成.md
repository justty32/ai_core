# 01 · 語言速成

Common Lisp 也是 `(函式 參數...)`。以下每段都在 SBCL 2.6.6 上跑過。

> 跟 Janet / Scheme 不同：CL **只有 `()`**（沒有內建的 `[]`/`{}` 資料語法）。
> vector 用 `#(...)`、hash-table 用建構函式，見 [02-資料結構.md](02-資料結構.md)。

## 綁定

```lisp
(let  ((a 1) (b 2)) (+ a b))      ; => 3   平行綁定
(let* ((a 1) (b (* a 10))) b)     ; => 10  循序綁定，b 看得到 a
(defparameter *g* 42)             ; 全域（慣例用 *earmuffs*）
(defvar *once* 1)                 ; 同上，但已綁定就不覆寫
```

`let` 的綁定同時發生（b 看不到剛綁的 a）；要看得到就用 `let*`。

## 函式

```lisp
(defun add (a b) (+ a b))                    ; (add 3 4) => 7
(funcall (lambda (x) (* x x)) 5)             ; => 25   匿名函式要 funcall
(defun greet (&optional (who "world"))       ; 可選參數 + 預設值
  (format nil "Hi, ~a" who))                 ; (greet) => "Hi, world"
(defun total (&rest xs) (reduce #'+ xs))     ; 變長參數 (total 1 2 3 4) => 10
```

`#'name` 是「取函式物件」（function 命名空間跟變數分開，這是 CL 的 Lisp-2 特性）。
高階函式要 `#'`：`(mapcar #'add ...)`、`(funcall #'add 1 2)`。

## 條件

```lisp
(if (> 3 2) "big" "small")        ; => "big"     只有一真一假
(when  (> 3 2) (do-a) (do-b))     ; 真才做、可多句
(unless (> 3 2) (do-c))           ; 假才做
(cond ((< 3 0) "neg")             ; 多分支
      ((= 3 0) "zero")
      (t "pos"))                  ; => "pos"    t 是 else
(case 2 (1 "one") (2 "two") (t "other"))   ; => "two"  值比對
```

真假值：只有 `nil`（＝空串列 `'()`）是假，其它全部是真（含 `0`、`""`）。真的正典值是 `t`。

## loop（CL 的瑞士刀迴圈）

```lisp
(loop for i from 0 below 3 collect i)           ; => (0 1 2)
(loop for x in '(1 2 3) collect (* x x))        ; => (1 4 9)
(loop for i from 1 to 5 when (oddp i) collect i); => (1 3 5)
(loop for x in '(1 2 3) do (print x))           ; 副作用
(loop repeat 3 do (princ "x"))                  ; xxx
```

`collect` 收集成串列、`sum`/`count`/`maximize` 做聚合、`do` 純副作用、`when`/`unless` 過濾。
也有簡單的 `dolist` / `dotimes`：

```lisp
(dolist (x '(:a :b)) (print x))
(dotimes (i 3) (print i))         ; 0 1 2
```

## format（輸出格式化，指令很多）

```lisp
(format t   "~a~%" x)     ; t = 印到 stdout；~a = 給人看；~% = 換行
(format nil "~a" x)       ; nil = 回傳字串（不印）
(format t "~s" "hi")      ; ~s = 可讀（給機器/含引號）=> "hi"
(format t "~d / ~,2f" 5 3.14159)   ; ~d 整數、~,2f 兩位小數
(format t "~{~a ~}" '(1 2 3))      ; ~{...~} 走訪串列 => 1 2 3
```

`~a` 幾乎是你最常用的；要回傳字串而不是印出來就 `(format nil ...)`。

## 註解與 docstring

```lisp
; 單分號到行尾（行內）
;;; 三分號慣例用於檔案/段落層級註解
(defun f (x)
  "第一個字串是 docstring，(documentation 'f 'function) 看得到。"
  x)
```
