В§10.1 protocol/ вҖ”вҖ” function дҪңиҖ…иҲҮ manager дҪңиҖ…зҡ„е…ұз”Ё helper

src/ai_core/protocol/ жҳҜе…§йғЁ helper зҡ„йӣҶдёӯеҮәеҸЈгҖӮжүҖжңү function дҪңиҖ…пјҲеҗ« ai_core иҮӘиә«е…ғд»¶пјүеңЁеҜҰдҪң --metadataгҖҒ--json-errorsгҖҒtoken иҷ•зҗҶгҖҒqueue / rate-limit / graceful shutdown зӯүжЁЈжқҝжҷӮпјҢйғҪжҮүи©Іе„Әе…Ҳ import жӯӨжЁЎзө„иҖҢйқһиҮӘе·ұйҮҚжҠ„гҖӮ

иЁӯиЁҲеӢ•ж©ҹпјҲеҮәиҮӘ thinking.md з¬¬ 4вҖ“5 иЎҢпјүпјҡ

гҖҢд»ҘеҫҢеңЁиЈҪдҪң function жҷӮпјҢиҷ•зҗҶйӮЈдәӣ metadata зҡ„еёёз”Ёж“ҚдҪңеҰӮжӘўжҹҘжҹҗеҖӢ key еӯҳдёҚеӯҳеңЁд№ӢйЎһпјҢеҸҜд»Ҙеј„жҲҗдёҖеҖӢ python module ж–№дҫҝдҪҝз”ЁгҖӮserver д№ӢйЎһд№ҹжҳҜгҖӮгҖҚ

еӯҗжЁЎзө„ е·ІеӯҳеңЁ е…§е®№ дё»иҰҒжңҚеӢҷе°ҚиұЎ
metadata.py вң… fetch_metadata(path) -> MetadataViewгҖҒmake_json_error / print_json_error / should_use_json_errorsпјҲdoc_11_arch_protocol.md В§4.6 / В§5.1пјү жүҖжңү functionгҖҒhubгҖҒcaller
env.py вң… е…Ёзі»зөұе…ұз”Ёз’°еўғи®Ҡж•ёеҗҚзЁұеёёж•ёпјҲAI_CORE_TOKEN / AI_CORE_FUNCS_DIR / вҖҰпјү жүҖжңүе…ғд»¶
server.py вҸі иҰҸеҠғ Singleton Resource Manager Pattern зҡ„ server жЁЈжқҝпјҡtoken з”ўз”ҹ / и®ҖжӘ”пјҲdoc_12_arch_entry_manager.md В§6.13пјүгҖҒFastAPI app е·Ҙе» гҖҒgraceful shutdownгҖҒ--metadata / --entry-metadata з«Ҝй»һйӘЁжһ¶ д»»дҪ• server-backed functionгҖҒз¬¬дәҢеҖӢд»ҘеҫҢзҡ„ singleton manager
queue.py вҸі иҰҸеҠғ йҖҡз”Ё per-entry FIFO queue + worker + time_to_wait_ms йҖҫжҷӮиҷ•зҗҶпјҲеҫһ entry_manager/queue.py жҠҪйӣўйҖҡз”ЁйғЁеҲҶпјү д»»дҪ• entry-managing singleton
ratelimit.py вҸі иҰҸеҠғ rpm / tokens / cost дёүи»ёйҷҗйЎҚзҡ„йҖҡз”ЁеҜҰдҪңпјҲеҫһ entry_manager/ratelimit.py жҠҪйӣўйҖҡз”ЁйғЁеҲҶпјү еҗҢдёҠ

жҠҪеҸ–жҷӮж©ҹпјҲYAGNIпјүпјҡвҸі иҰҸеҠғдёӯзҡ„дёүй …дёҚеңЁз¬¬дёҖзүҲеҒҡпјҢиҖҢжҳҜзӯүз¬¬дәҢеҖӢ singleton manager еҮәзҸҫжҷӮжүҚеҫһ entry_manager/ жҠҪеҮәпјҢйҒҝе…ҚйҒҺж—©жҠҪиұЎгҖӮдҪҶзӣ®йҢ„дҪҚзҪ®е…Ҳй җз•ҷпјҢdocstring д№ҹжҢҮеҗ‘жӯӨиҷ•пјҢи®“жңӘдҫҶжҠҪеҸ–жҷӮдёҚйңҖиҰҒж”№ import path еӨ§жүӢиЎ“гҖӮ

е°Қ function дҪңиҖ…зҡ„жүҝи«ҫпјҡ

  • protocol/ зҡ„ API дёҖж—Ұз©©е®ҡе°ұзӣЎйҮҸдёҚз ҙеЈһжҖ§и®Ҡжӣҙ
  • жүҖжңү helper йғҪе…ҒиЁұ graceful йҷҚзҙҡпјҲзјәж¬„дҪҚз”ЁеҗҲзҗҶй җиЁӯгҖҒдёҚдёҹдҫӢеӨ–зөҰ callerпјүпјҢиҲҮ doc_11_arch_protocol.md В§4.6 / В§5 зҡ„е®№йҢҜеҺҹеүҮдёҖиҮҙ
  • function дҪңиҖ…дёҚйңҖиҰҒд№ҹдёҚжҮүи©І import entry_manager/ / hub/ / е…¶д»–е…ғд»¶е…§йғЁпјҢеҸӘжҮүдҪҝз”Ё protocol/
