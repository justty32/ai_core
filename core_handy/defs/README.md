# defs/ — 描述性定義層

對應 `notes/00_index.md`「兩層區分」的第 1 類內容：**描述性 metadata 型別**。

這裡放的是「描述一個函數**是什麼**」的型別——九軸標準定義的 C++ 濃縮。純資料、純描述，
不含「替程式做事」的功能碼（那屬於 [`../impl/`](../impl/)）。

對應軸（描述性）：
- 軸 1 `entries`（I/O 出入口）
- 軸 2 `lifecycle`（生命週期）
- 軸 3 `state`（`bool stateful`）
- 軸 5–9（待討論）

> 軸 4 `state_dirs` **不在此層**——它在標準定義裡不存在（程式狀態自管），只在實作庫中以設施形式提供，見 `../impl/`。
